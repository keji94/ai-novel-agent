# 核心角色群

---

## 一、盟友阵营

### 1. 老孙头（孙德福）
- **身份**：拾荒者老手，苏辰的养父
- **年龄**：60多岁（末世前就已是中年）
- **性格**：表面圆滑市侩，内心善良仗义；嘴碎爱唠叨；对苏辰视如己出
- **能力**：一阶触物者，共鸣物是一把破旧的洛阳铲（可感知地下古物位置）
- **隐藏过去**：老孙头曾是长安城共鸣者公会的成员，因一次任务中违抗上级"放弃平民"的命令而被除名，自我放逐到三环外。他刻意留在废墟中，是因为当年救出婴儿苏辰后，发现这孩子身份特殊（血裔混血），留在三环外反而最安全——这里信息闭塞、无人关注。他隐藏共鸣者身份装成普通拾荒者，十年如一日地保护苏辰，直到异族袭击时不得不暴露实力
- **角色定位**：亲情线核心 / 前期引路人 / 牺牲flag / 伏笔人物
- **关键剧情**：
  - 第一卷中期为保护苏辰被异族杀害 → 成为主角彻底觉醒的催化剂
  - 死前将一个"铜匣子"交给苏辰（内含苏辰身世线索）
  - 死亡时暴露真实实力（远超一阶），暗示其过去不简单，为后续揭秘留口
- **经典台词**："小辰啊，你记着，在这废墟里活着，比什么都强。别逞英雄，英雄都死绝了。"

### 2. 林晚棠
- **身份**：长安城二环共鸣者公会的新人，二阶融物者
- **年龄**：18岁
- **外貌**：短发利落，眼神倔强，身上总带着一股不服输的劲
- **性格**：正义感强但天真，不服输，嘴硬心软
- **共鸣物**：一把宋代团扇（可释放风刃+操控气流）
- **角色定位**：女主角 / 代表"体制内"视角 / 与主角形成互补
- **成长弧线**：天真热血 → 被现实打击 → 重新找到信念 → 成为并肩作战的战友 → 独立政治力量
- **个人目标与独立线**：
  - 核心目标：证明"规则"和"正义"在末世仍然有意义，而非空话
  - 第二卷独立线：违抗沈墨寒的清洗令，暗中保护被怀疑的无辜者，从"主角的跟班"成长为有独立判断力的行动者。在清洗风波中建立自己的情报网络
  - 第三卷独立线：在苏辰流放期间，林晚棠独自在长安城内维持公会的正义底线，成为基层共鸣者信赖的领袖。长安保卫战中，她不是"为主角而来"，而是"为保护长安城而来"——动机独立
- **与主角关系**：从误解对立→互相欣赏→并肩作战→暧昧（但两人各有独立立场，非完全绑定）
- **关键剧情**：
  - 第一卷初登场时因任务到三环外，与苏辰产生冲突（误解他是异族探子）
  - 第二卷因主角混血身份曝光而陷入两难
  - 第三卷选择相信主角，共同面对终极危机

### 3. 铁牛（真名：赵铁牛）
- **身份**：三环外拾荒者小队的头领
- **年龄**：25岁
- **外貌**：身材魁梧如熊，脸上有一道从额头到下巴的旧疤
- **性格**：豪爽粗犷，讲究义气，有勇无谋但听劝
- **反差面**：外表粗犷但内心细腻——他会偷偷攒食物分给拾荒者营地里的孤儿；他的旧疤是在保护一个陌生拾荒者时留下的（从不主动提起）；识字（在末世很罕见），会在休息时用树枝在泥地上练字，梦想有一天能"读懂博物馆里的书"。这种反差让他在粗犷的"莽汉"形象下有着令人意外的温柔和向往
- **能力**：未共鸣（前期），后期与一面破损的盾牌共鸣，成为一阶触物者
- **角色定位**：主角第一个真正的兄弟 / 肉盾担当 / 喜剧调节
- **关键剧情**：
  - 前期看不起独臂的苏辰，后被苏辰的智谋折服
  - 第二卷成为主角小队的核心战力
- **经典台词**："跟着苏辰兄弟混，吃肉！不跟着……也饿不死，但没肉吃。"

### 4. 白老先生（白慎行）
- **身份**：长安城核心区域的隐居者，五阶通灵者
- **年龄**：70多岁（实际年龄不详，通灵者寿命更长）
- **性格**：慈祥中透着锐利，说话喜欢打哑谜，但关键时刻一针见血
- **共鸣物**：一支毛笔（据说是王羲之遗物残片），可书写"言灵"——写下的话会成真（有限制）
- **角色定位**：导师型角色 / 长安城隐藏的大佬 / 知道苏辰身世的关键人物
- **关键剧情**：
  - 第一卷末暗中出手救了苏辰一命
  - 第二卷中揭露苏辰身世第一层真相
  - 第三卷中传授苏辰关键能力
- **经典台词**："年轻人，你知道为什么古物会选择人类吗？因为只有人类，会为了一段记忆去拼命。"

## 二、对手阵营

### 5. 沈墨寒
- **身份**：长安城共鸣者公会会长的养子，四阶御物者
- **年龄**：22岁
- **外貌**：清俊儒雅，永远面带微笑，眼底却毫无温度
- **性格**：表面温文尔雅，实际冷酷自私；极度理性，将人视为棋子
- **共鸣物**：多件古物（剑、镜、印），可切换使用
- **角色定位**：前期主要对手 / 代表体制内的腐败与阶级压迫
- **动机**：认为只有强者配生存，弱者是累赘；对苏辰从无视→忌惮→欲除之
- **关键剧情**：
  - 第一卷为争夺古物与苏辰产生冲突
  - 第二卷利用苏辰混血身份煽动舆论
  - 第二卷末转变触发事件：异族大军压境时，沈墨寒的养父（公会会长）在守城战中重伤昏迷，临危托付长安城给沈墨寒。这个责任让他第一次面对"个人权力"与"守护家园"的抉择——他可以选择带着核心资源逃跑，但最终选择留下守城。这个转变不是突然的"洗白"，而是被逼到绝境后的选择
  - 第三卷末展现出复杂一面（选择守护长安城）
- **经典台词**："这个世界从来就不公平。你以为你是在反抗命运？不，你只是在证明——你比我想象的更危险。"

### 6. 柳青衣
- **身份**：匠造协会的天才少女，三阶化物者+匠造大师
- **年龄**：20岁
- **外貌**：面容精致，总是穿一身青色工装，手上永远有墨渍
- **性格**：痴迷炼制，对其他事漠不关心；社交障碍，说话直来直去
- **角色定位**：技术流对手/盟友 / 与主角的造化炉形成对比和互补
- **与主角关系**：从竞争（争夺古物材料）→合作（发现主角的炼制天赋）→友谊
- **关键剧情**：
  - 第一卷末与主角比试炼制，输了但因此对主角产生兴趣
  - 第二卷协助主角修复关键古物
- **经典台词**："你的炼制方法……不可能。没有任何文献记载过这种手法。你到底是谁？"

## 三、反派阵营

### 7. 赫连昭（血裔族·暗影君王）
- **身份**：血裔族高层之一，超越人类认知的存在（对应人类体系约六阶以上），苏辰的……父亲
- **外貌**：人形时为俊美中年男子，战斗时展露血裔真身——半人半蝠
- **性格**：冷漠但有其扭曲的"父爱"；对人类的看法复杂（曾爱上一个人类女性）
- **角色定位**：终极反派 / 身世谜团的核心 / 不是纯粹的恶
- **动机**：
  - 表面：追杀苏辰（因为苏辰是"钥匙"，不能落入人类手中）
  - 政治压力层面：血裔族高层分为"主杀派"和"利用派"。赫连昭作为暗影君王必须在两派之间平衡——如果他公开庇护混血儿子，会被主杀派视为叛族；如果他真的杀了苏辰，就永远失去了"钥匙"的潜在价值。所以他采取"派人试探施压"而非"直接追杀"的策略，既向主杀派表态，又在暗中为苏辰创造成长空间
  - 深层：想让苏辰觉醒血脉，继承他的力量，以血裔族的"方式"保护他
  - 最深层：他当年爱上苏辰母亲，斩断苏辰手臂封印血脉的命令……也是他下的
- **关键剧情**：
  - 第一卷通过手下追杀苏辰（伏笔）
  - 第二卷末正式登场，揭露身世真相
  - 第三卷末与苏辰最终对决
- **经典台词**："你恨我？很好。恨比遗忘更有力量。你的母亲教我的。"

### 8. 赫连夜的幻影（塑形族·渗透者）
- **身份**：塑形族高级间谍，潜入长安城多年
- **外貌**：可变，长期以"韩夜"身份活动——一个普通的二环商贩
- **角色定位**：中期反派 / 制造内部信任危机
- **关键剧情**：
  - 第二卷中段身份暴露，造成长安城内部恐慌
  - 与苏辰的对决揭示塑形族的弱点
  - 死前透露异族入侵长安城的时间表
