# 核心角色群

---

## 一、盟友阵营

### 1. 老孙头（孙德福）
- **身份**：拾荒者老手，苏辰的养父
- **年龄**：60多岁（末世前就已是中年）
- **性格**：表面圆滑市侩，内心善良仗义；嘴碎爱唠叨；对苏辰视如己出
- **能力**：一阶触物者，共鸣物是一把破旧的洛阳铲（可感知地下古物位置）
- **角色定位**：亲情线核心 / 前期引路人 / 牺牲flag
- **关键剧情**：
  - 第一卷中期为保护苏辰被异族杀害 → 成为主角彻底觉醒的催化剂
  - 死前将一个"铜匣子"交给苏辰（内含苏辰身世线索）
- **经典台词**："小辰啊，你记着，在这废墟里活着，比什么都强。别逞英雄，英雄都死绝了。"

### 2. 林晚棠
- **身份**：长安城二环共鸣者公会的新人，二阶融物者
- **年龄**：18岁
- **外貌**：短发利落，眼神倔强，身上总带着一股不服输的劲
- **性格**：正义感强但天真，不服输，嘴硬心软
- **共鸣物**：一把宋代团扇（可释放风刃+操控气流）
- **角色定位**：女主角 / 代表"体制内"视角 / 与主角形成互补
- **成长弧线**：天真热血 → 被现实打击 → 重新找到信念 → 成为并肩作战的战友
- **与主角关系**：从误解对立→互相欣赏→并肩作战→暧昧
- **关键剧情**：
  - 第一卷初登场时因任务到三环外，与苏辰产生冲突（误解他是异族探子）
  - 第二卷因主角混血身份曝光而陷入两难
  - 第三卷选择相信主角，共同面对终极危机

### 3. 铁牛（真名：赵铁牛）
- **身份**：三环外拾荒者小队的头领
- **年龄**：25岁
- **外貌**：身材魁梧如熊，脸上有一道从额头到下巴的旧疤
- **性格**：豪爽粗犷，讲究义气，有勇无谋但听劝
- **能力**：未共鸣（前期），后期与一面破损的盾牌共鸣，成为一阶触物者
- **角色定位**：主角第一个真正的兄弟 / 肉盾担当 / 喜剧调节
- **关键剧情**：
  - 前期看不起独臂的苏辰，后被苏辰的智谋折服
  - 第二卷成为主角小队的核心战力
- **经典台词**："跟着苏辰兄弟混，吃肉！不跟着……也饿不死，但没肉吃。"

### 4. 白老先生（白慎行）
- **身份**：长安城核心区域的隐居者，五阶通灵者
- **年龄**：70多岁（实际年龄不详，通灵者寿命更长）
- **性格**：慈祥中透着锐利，说话喜欢打哑谜，但关键时刻一针见血
- **共鸣物**：一支毛笔（据说是王羲之遗物残片），可书写"言灵"——写下的话会成真（有限制）
- **角色定位**：导师型角色 / 长安城隐藏的大佬 / 知道苏辰身世的关键人物
- **关键剧情**：
  - 第一卷末暗中出手救了苏辰一命
  - 第二卷中揭露苏辰身世第一层真相
  - 第三卷中传授苏辰关键能力
- **经典台词**："年轻人，你知道为什么古物会选择人类吗？因为只有人类，会为了一段记忆去拼命。"

## 二、对手阵营

### 5. 沈墨寒
- **身份**：长安城共鸣者公会会长的养子，四阶御物者
- **年龄**：22岁
- **外貌**：清俊儒雅，永远面带微笑，眼底却毫无温度
- **性格**：表面温文尔雅，实际冷酷自私；极度理性，将人视为棋子
- **共鸣物**：多件古物（剑、镜、印），可切换使用
- **角色定位**：前期主要对手 / 代表体制内的腐败与阶级压迫
- **动机**：认为只有强者配生存，弱者是累赘；对苏辰从无视→忌惮→欲除之
- **关键剧情**：
  - 第一卷为争夺古物与苏辰产生冲突
  - 第二卷利用苏辰混血身份煽动舆论
  - 第三卷末在异族大举入侵时展现出复杂一面（选择守护长安城）
- **经典台词**："这个世界从来就不公平。你以为你是在反抗命运？不，你只是在证明——你比我想象的更危险。"

### 6. 柳青衣
- **身份**：匠造协会的天才少女，三阶化物者+匠造大师
- **年龄**：20岁
- **外貌**：面容精致，总是穿一身青色工装，手上永远有墨渍
- **性格**：痴迷炼制，对其他事漠不关心；社交障碍，说话直来直去
- **角色定位**：技术流对手/盟友 / 与主角的造化炉形成对比和互补
- **与主角关系**：从竞争（争夺古物材料）→合作（发现主角的炼制天赋）→友谊
- **关键剧情**：
  - 第一卷末与主角比试炼制，输了但因此对主角产生兴趣
  - 第二卷协助主角修复关键古物
- **经典台词**："你的炼制方法……不可能。没有任何文献记载过这种手法。你到底是谁？"

## 三、反派阵营

### 7. 赫连昭（血裔族·暗影君王）
- **身份**：血裔族高层之一，七阶级强者，苏辰的……父亲
- **外貌**：人形时为俊美中年男子，战斗时展露血裔真身——半人半蝠
- **性格**：冷漠但有其扭曲的"父爱"；对人类的看法复杂（曾爱上一个人类女性）
- **角色定位**：终极反派 / 身世谜团的核心 / 不是纯粹的恶
- **动机**：
  - 表面：追杀苏辰（因为苏辰是"钥匙"，不能落入人类手中）
  - 深层：想让苏辰觉醒血脉，继承他的力量，以血裔族的"方式"保护他
  - 最深层：他当年爱上苏辰母亲，斩断苏辰手臂封印血脉的命令……也是他下的
- **关键剧情**：
  - 第一卷通过手下追杀苏辰（伏笔）
  - 第二卷末正式登场，揭露身世真相
  - 第三卷末与苏辰最终对决
- **经典台词**："你恨我？很好。恨比遗忘更有力量。你的母亲教我的。"

### 8. 赫连夜的幻影（塑形族·渗透者）
- **身份**：塑形族高级间谍，潜入长安城多年
- **外貌**：可变，长期以"韩夜"身份活动——一个普通的二环商贩
- **角色定位**：中期反派 / 制造内部信任危机
- **关键剧情**：
  - 第二卷中段身份暴露，造成长安城内部恐慌
  - 与苏辰的对决揭示塑形族的弱点
  - 死前透露异族入侵长安城的时间表
