# 主线剧情大纲

## 整体规划
- 预计字数：200-300万字
- 卷数规划：3大卷 + 后续可扩展
- 核心主题：文明记忆的力量 / 人性的选择 / 底层逆袭
- 节奏基调：慢热开局→中期加速→后期爆发

---

## 第一卷：废墟拾荒者（约60-80章）

### 卷简介
苏辰穿越成三环外的独臂拾荒者少年，在末世废墟中挣扎求生。意外激活古物星河和造化炉，凭借前世博物馆知识迅速崛起。从无人问津的废物，成长为三环外的传奇人物，最终踏入长安城。

### 章节规划

**第1-5章：穿越与觉醒**
- 第1章：醒来。独臂少年在废墟中醒来，记忆融合。发现自己身处末世，周围是断壁残垣和异虫残骸。老孙头救了他。
- 第2章：拾荒。跟着老孙头学习废墟生存法则。第一次遭遇虫潮，差点丧命。展示独臂少年的艰难处境。
- 第3章：古物。在废墟深处发现一面铜镜，触碰瞬间精神识海中出现"星河"——古物星河激活。第一颗星点亮：汉代铜镜，信息涌入。
- 第4章：共鸣。尝试与铜镜共鸣，获得第一项能力"鉴真之眼"——可辨别古物真伪和价值。造化炉也同步激活，但功能有限。
- 第5章：初战。利用对铜镜的了解，在拾荒者争夺中智取一件被忽视的古物（别人当垃圾的东西他看出价值）。第一次展现前世知识的碾压。

**第6-15章：立足三环外**
- 第6-8章：铁牛登场。拾荒者小队冲突，苏辰用计化解危机，赢得铁牛尊重。建立初步团队。
- 第9-10章：深入废墟。发现一处末世前的博物馆遗址（主角前世工作的博物馆同款！）。情绪爆发但迅速恢复冷静。博物馆内大量古物，成为苏辰的"宝库"。
- 第11-12章：古物星河大丰收。一口气点亮十几颗凡星。造化炉升级，解锁合成功能。苏辰开始秘密炼制物品提升实力。
- 第13-15章：第一次共鸣突破。从一阶触物者突破到二阶融物者。独臂的限制反而让他精神力更集中。与一件宋代瓷瓶共鸣，获得"瓷盾"能力。

**第16-25章：三环外风云**
- 第16-18章：林晚棠登场。公会任务到三环外，与苏辰产生冲突。误解→化解→初步了解。苏辰帮她解决一个难题（用古物知识辨别出塑形族伪装的低级间谍），赢得信任。
- 第19-21章：拾荒者大战。三环外多个拾荒者团伙为争夺一处新发现的古建筑废墟而火拼。苏辰用计整合势力，成为三环外最强拾荒者势力领袖。
- 第22-25章：老孙头之死。异族袭击三环外区域，老孙头为掩护撤退牺牲。死前交给苏辰铜匣子。苏辰第一次展现真正的战斗力——暴走状态下击杀虫将。

**第26-35章：长安城初入**
- 第26-28章：进入三环。苏辰凭借实力和贡献获得进入三环的资格。看到长安城的繁华（相比三环外），也看到阶级分化。
- 第29-31章：共鸣者公会注册。参加入会考核，因独臂被轻视。考核中一鸣惊人——以二阶实力通过三阶考核标准。
- 第32-35章：沈墨寒登场。与沈墨寒因一次任务分配产生冲突。沈墨寒看出苏辰不简单，暗中打压。苏辰与柳青衣的首次比试（炼制对决），苏辰用造化炉险胜。

**第36-45章：暗流涌动**
- 第36-38章：铜匣子线索。打开铜匣子，发现一枚玉佩和一封信（残缺）。信中暗示苏辰的身世——"他的手臂不能留着"。
- 第39-41章：深入调查。苏辰开始追查身世。白老先生暗中观察，第一次与苏辰对话，给出模棱两可的提示。
- 第42-45章：三环外异变。异族活动异常频繁，似乎在寻找什么。苏辰在一次探索中遭遇塑形族（低级），第一次知道异族可以伪装成人。

**第46-55章：卷末高潮**
- 第46-48章：潮汐日。强源潮降临，长安城外的古物共鸣大幅增强。苏辰在潮汐日中与一件意外出土的古剑共鸣，突破三阶化物者。
- 第49-52章：虫潮来袭。大规模虫潮冲击长安城外围防线。苏辰主动请缨参与防御，在战斗中大放异彩。
- 第53-55章：三环保卫战。防线被突破，三环区域危急。苏辰带领三环外的拾荒者自发组织防御。沈墨寒想放弃三环，苏辰阻止。白老先生暗中出手。最终守住。

**第56-60章：卷末收束**
- 第56-58章：身份伏笔。战后，苏辰的战斗方式引起了一些人的注意——他的某些能力"不像是纯人类的共鸣"。赫连夜的幻影在暗中观察。
- 第59-60章：踏入二环。因保卫战功绩，苏辰获得进入二环的资格。第一卷结尾：苏辰站在二环的入口，身后是三环外无数仰望的目光，前方是未知的挑战。铜匣子中的玉佩微微发热……

### 本卷核心爽点
1. 古物知识碾压：别人当垃圾的古物他一眼看出价值
2. 造化炉炼制：用废料炼出神器级物品
3. 独臂逆袭：以残躯击败完壮对手
4. 老孙头之死→暴走→击杀虫将（情绪爽点）
5. 入会考核一鸣惊人
6. 三环保卫战高光

### 伏笔埋设
- 铜匣子和玉佩（身世线）
- 白老先生的身份和目的
- 苏辰战斗方式中的"异常"
- 赫连夜的暗中观察
- 沈墨寒的复杂态度
- 造化炉的来源之谜

## 第二卷：长安暗影（约60-80章）

### 卷简介
苏辰进入二环，在共鸣者公会中崭露头角。身世谜团逐渐揭开——他竟是人族与血裔族的混血。身份曝光引发轩然大波，被追杀、被质疑、被利用。在困境中，苏辰走出自己的路。

### 章节规划

**第1-10章：二环立足**
- 第1-3章：二环的规则。公会内部派系林立，苏辰被分到最边缘的"外勤队"。结识新队友。
- 第4-6章：古物交易。苏辰利用造化炉炼制物品，在商会中赚取资源。第一次接触匠造协会，与柳青衣深入交流。
- 第7-10章：首次深入任务。公会组织探索长安城外的一处大型古建筑遗址（疑似唐代宫殿遗址）。苏辰的古物知识再次大放异彩。

**第11-20章：身世揭秘（第一层）**
- 第11-13章：玉佩的秘密。苏辰找到白老先生，要求解答玉佩的疑问。白老先生带他到长安城地下——一个保存完好的末世前档案馆。
- 第14-16章：真相碎片。档案馆中的记录揭示：47年前，有一个独臂婴儿被一位五阶共鸣者从血裔族手中救出。婴儿的左臂被切断，封印了"某种东西"。
- 第17-20章：追查生父。苏辰发现自己的"母亲"曾在长安城核心区域居住过，但在他很小的时候就"消失"了。线索指向核心区域的高层。

**第21-30章：塑形族渗透**
- 第21-23章：异常事件。长安城内连续发生怪事——有人失踪、有人行为反常。苏辰怀疑是塑形族。
- 第24-27章：赫连夜暴露。苏辰通过古物星河的感知能力，发现"商贩韩夜"是塑形族。一场激烈的对决，塑形族间谍被消灭，但临死透露：异族大军即将入侵长安城。
- 第28-30章：内部恐慌。间谍事件引发长安城内部信任危机。人人自危。沈墨寒借机扩大权力，提议"清洗"所有有嫌疑的人。苏辰反对，与沈墨寒彻底对立。

**第31-40章：身世曝光**
- 第31-33章：混血真相。在一次战斗中，苏辰受伤后左臂断口处涌出血裔族的红色能量——混血身份曝光。
- 第34-36章：众叛亲离。长安城内部震动。有人要求处死苏辰（认为他是异族探子），有人沉默。林晚棠陷入两难。铁牛选择站在苏辰一边。
- 第37-40章：流放。白老先生在关键时刻发声保下苏辰，但苏辰被"流放"出长安城。柳青衣暗中给予帮助。沈墨寒如释重负。

**第41-50章：废墟中的修行**
- 第41-43章：重新开始。苏辰回到三环外废墟，但这次他有目标——掌控自己的力量。
- 第44-46章：血脉与共鸣。苏辰尝试同时运用血裔血脉和古物共鸣，两股力量冲突剧烈。在生死边缘找到平衡——古物共鸣压制血脉，同时利用血脉的能量增强共鸣。
- 第47-50章：第四阶突破。在废墟深处发现一件古物——一把断裂的青铜剑，竟是传说中的"越王勾践剑"残片。与残片共鸣，突破四阶御物者。造化炉修复残片成功。

**第51-60章：卷末高潮**
- 第51-53章：异族大军压境。赫连夜临死前透露的情报成真——血裔族大军直指长安城。这是血裔族寻找"钥匙"的行动。
- 第54-56章：赫连昭登场。血裔族暗影君王亲自率军。在战场上与苏辰对视——父子相认。
- 第57-60章：长安保卫战。长安城存亡之际，苏辰以"独臂的混血"身份重返战场。铁牛带着三环外的拾荒者自发支援。林晚棠违抗命令出城参战。沈墨寒也展现出守护长安城的一面。
- 苏辰与赫连昭的第一次正面交锋。赫连昭试探性地出手，发现苏辰的力量已经超出预期。战斗中赫连昭说出了关键信息："你的母亲……选择了你。"

### 本卷核心爽点
1. 身世揭秘的震撼感
2. 众叛亲离后绝地翻盘
3. 血脉+共鸣双修的爽感
4. 长安保卫战中王者归来
5. 与赫连昭的父子对决
6. 越王勾践剑残片的获得

### 伏笔埋设
- 赫连昭说的"选择了你"是什么意思？
- 钥匙的具体作用
- 苏辰母亲的真正死因
- 造化炉的来源（开始有线索）

## 第三卷：文明之战（约60-80章）

### 卷简介
异族入侵只是前奏，真正的危机是"裂隙"的全面开启。苏辰的身份——"钥匙"——被彻底揭露。他必须在"觉醒血脉成为血裔"和"以人类身份对抗命运"之间做出选择。最终，古物共鸣的力量证明：文明的记忆，才是最强的武器。

### 章节规划

**第1-15章：裂隙危机**
- 第1-5章：战后重建。长安城虽守住但元气大伤。苏辰被重新接纳，但仍有很多人不信任他。开始调查裂隙全面开启的原因。
- 第6-10章：全球形势。其他聚集地的消息传来——世界各地的裂隙都在扩大。这不是针对长安城，而是全球性的异族总攻。
- 第11-15章：钥匙的真相。白老先生终于全盘托出：裂隙需要"钥匙"才能完全打开或永久关闭。苏辰是唯一的钥匙——因为只有同时拥有文明共鸣力和血裔血脉的存在才能影响裂隙。

**第16-30章：寻找方法**
- 第16-20章：古物星河升级。苏辰的古物星河在战斗中进一步觉醒，点亮了第一颗王星（金色）——一件传说中的古物信息。
- 第21-25章：造化炉的来历。造化炉的真相揭晓——它不是外挂，而是末世前一位六阶造化者留下的遗物。造化者预见了末世，将毕生所学封入造化炉，等待"合适的人"使用。苏辰的前世博物馆经历让他成为那个人。
- 第26-30章：第三条路。苏辰拒绝"觉醒血脉"和"放弃力量"的两个选项，决定走出第三条路——用古物共鸣的力量融合血脉，以人类的文明之力关闭裂隙。白老先生决定全力辅助。

**第31-45章：终极备战**
- 第31-35章：集结盟友。苏辰前往其他聚集地寻求支援。途中遇到各种人物和势力，展现末世全貌。
- 第36-40章：五阶突破。在旅途中与一件商代青铜鼎共鸣，突破五阶通灵者。获得"鼎镇山河"的能力——可在范围内增强所有友方的古物共鸣。
- 第41-45章：最终决战准备。各路英雄汇聚长安城。苏辰制定了一个疯狂的计划——不是防守，而是主动进入裂隙，从内部关闭它。

**第46-60章：最终决战**
- 第46-50章：裂隙之战。人类联军在裂隙外与异族大军决战。苏辰带领精锐小队突入裂隙。
- 第51-55章：裂隙内部。裂隙内部是一个扭曲的空间，充满了异族的本源力量。苏辰一行人遭遇血裔族最强战力。
- 第56-58章：父子最终对决。苏辰与赫连昭的终极一战。赫连昭的目的是让苏辰觉醒血脉，加入血裔族。苏辰拒绝，以古物共鸣之力+血脉能量，走出第三条路。
- 第59章：裂隙关闭。苏辰用自身为媒介，以古物星河中所有点亮的古物之力为燃料，强行关闭裂隙。代价：古物星河暂时熄灭，力量大幅衰退。
- 第60章：新的开始。裂隙关闭，异族失去入侵通道。苏辰重伤昏迷。醒来后发现自己失去了大部分力量，但古物星河的星星正在一颗一颗重新亮起——新的古物，新的故事，新的冒险。

### 本卷核心爽点
1. 全球视角的宏大战争
2. 古物星河点亮王星的震撼
3. 五阶通灵者的逆天能力
4. 主动出击的疯狂计划
5. 父子终极对决的情感爆发
6. 第三条路——文明共鸣融合血脉
7. 结局——星星重新亮起（留白+希望）

### 伏笔收束
- 苏辰身世全部揭晓 ✓
- 造化炉来历 ✓

- 赫连昭的复杂动机 ✓
- 古物星河的终极形态（暗示：星星可以无限点亮）
