# 世界观设定：末世古物共鸣

## 基本信息
- 作品类型：末世废土 + 异能进化
- 核心卖点：古物共鸣觉醒体系 / 底层逆袭 / 博物馆保安的文物知识碾压
- 整体风格：硬核求生、智斗布局、热血燃点
- 末世时间线：异变至今约47年（主角穿越时）

---

## 一、世界背景

### 异变纪元（纪元前0年）
- 一颗来历不明的陨石坠入太平洋，未造成直接破坏
- 但陨石释放的未知能量波及全球，被称为"源潮"
- 源潮持续三年，地球生物基因被深度改写

### 灾厄降临（纪元1-5年）
- 异族从"裂隙"中入侵，裂隙出现在古文明遗址附近
- 异族目标明确：灭绝人类，占领地球
- 人类文明在五年内崩塌，全球人口从80亿骤降至不足5亿
- 现代科技大量失效（源潮干扰电子设备），人类退回冷兵器时代

### 人类复兴（纪元5-20年）
- 少数人类发现与古物（古建筑、古董、古兵器等）产生共鸣可觉醒超凡力量
- 古建筑在源潮催化下产生保护力场，成为人类最后的堡垒
- 聚集地建立，以古建筑为核心向外扩张

### 当前时代（纪元47年）
- 人类已建立数百个聚集地，最大者可容纳百万人
- 但异族从未停止入侵，人类仍处于生死存亡的边缘
- 信息断层严重：末世前的大部分知识已失传
- 主角穿越的节点

---

## 二、古物共鸣体系

### 核心规则
- **共鸣条件**：人需与古物产生精神共振，需对古物有"理解"和"感悟"
- **古物定义**：年满50年以上且承载人类文明记忆的器物、建筑、遗迹
- **共鸣媒介**：古物的"文明记忆"——它所承载的历史、故事、情感
- **共鸣代价**：过强共鸣会导致精神侵蚀（古物中的执念、怨念反噬）

### 共鸣等级（共鸣者）

| 等级 | 称号 | 共鸣深度 | 能力特征 | 人数占比 |
|------|------|----------|----------|----------|
| 一阶 | 触物者 | 表面共鸣 | 获得古物一项特性 | 70% |
| 二阶 | 融物者 | 浅层共鸣 | 古物特性强化+外放 | 20% |
| 三阶 | 化物者 | 深度共鸣 | 可化用古物之力为招式 | 7% |
| 四阶 | 御物者 | 灵魂共鸣 | 可同时共鸣多件古物 | 2% |
| 五阶 | 通灵者 | 本源共鸣 | 与古物"对话"，解锁隐藏信息 | 0.8% |
| 六阶 | 造化者 | 创造共鸣 | 可赋予物品"古物属性" | 0.1% |
| 七阶 | 万物共鸣 | — | 传说境界，据说能与大地本身共鸣 | 仅存传说 |

### 共鸣类型

1. **武兵共鸣**：刀剑、铠甲、兵器 → 战斗力提升
2. **器物共鸣**：器皿、工具、乐器 → 辅助/特殊能力
3. **建筑共鸣**：古建筑、城池、桥梁 → 防御/领域能力
4. **书卷共鸣**：古籍、画卷、碑帖 → 知识/精神力
5. **遗物共鸣**：名人遗物、圣物 → 特殊/稀有能力

### 共鸣流派
- **武道派**：以兵器共鸣为主，强调战斗
- **匠造派**：以器物共鸣为主，擅长炼制
- **御城派**：以建筑共鸣为主，守护聚集地
- **文心派**：以书卷共鸣为主，精神系能力

## 三、异族设定

### 异族来源
- 来自裂隙彼岸的异世界种族
- 源潮是它们入侵的前奏，裂隙是它们的通道
- 它们以"吞噬文明"为食——越文明的种族，越美味

### 主要异族

**1. 虫潮族（最常见）**
- 外形：巨型昆虫，从蚂蚁到甲虫各变种
- 特点：数量庞大、繁殖极快、低级个体无智力
- 等级：工虫→兵虫→将虫→虫母
- 弱点：怕火、怕震

**2. 血裔族（高阶敌人）**
- 外形：人形但非人，皮肤苍白，瞳孔为红
- 特点：高智商、可操控低等异族、拥有"血术"
- 等级对应人类共鸣等级
- 弱点：阳光（但可适应）、古物共鸣之力

**3. 塑形族（特殊敌人）**
- 外形：可变形，能模仿人类
- 特点：渗透、间谍、制造内乱
- 极度危险：无法被常规手段识别
- 弱点：古物共鸣者可感知其"非人气息"

**4. 深渊巨兽（BOSS级）**
- 外形：各种巨型生物
- 特点：移动要塞级别，一次出现可毁灭聚集地
- 人类最强战力才能应对
- 弱点：核心（体内关键部位）

### 异族与古物的关系
- 异族本能地厌恶古物，因为古物承载的"文明记忆"对它们有天然克制
- 这也是人类能存活的关键原因
- 但高级异族已学会对抗古物之力

## 四、社会结构

### 聚集地结构

```
          ┌─────────┐
          │ 核心区域  │ ← 古建筑本体所在，权贵居住区
          │ （内城）  │    最强防御，资源最丰富
          └────┬────┘
               │
       ┌───────┴───────┐
       │   二环区域     │ ← 中产/共鸣者居住区
       │   （商住区）    │    有基础防御设施
       └───────┬───────┘
               │
       ┌───────┴───────┐
       │   三环区域     │ ← 底层/无共鸣者居住区
       │   （贫民区）    │    防御薄弱，随时可能被放弃
       └───────┴───────┘
               │
        ┌──────┴──────┐
        │  三环外废墟  │ ← 拾荒者、流浪者
        │  （荒野区）   │    异族出没，朝不保夕
        └─────────────┘
```

### 聚集地势力

**1. 共鸣者公会**
- 聚集地的实际掌控者
- 由高阶共鸣者组成
- 负责对外征伐、内部秩序
- 等级森严，低阶服从高阶

**2. 匠造协会**
- 炼制古物衍生品的专业组织
- 地位超然，各势力都需要的合作对象
- 内部有严格的师徒传承制度

**3. 商会**
- 掌控聚集地的经济命脉
- 收购古物、出售物资
- 有自己的武装力量

**4. 拾荒者**
- 三环外生存的人群
- 靠在废墟中搜寻古物和物资为生
- 主角的开局身份
- 社会地位最低，但最了解废墟

**5. 信仰教派**
- 崇拜古物/崇拜祖先的宗教组织
- 各聚集地都有不同教派
- 有的温和、有的极端

### 货币体系
- **源晶**：异族体内凝结的能量结晶，用途广泛
- **古物碎片**：古物碎裂后的残余，可用于低级共鸣催化
- 以物易物仍然普遍

### 核心聚集地（故事主要舞台）
- **长安城**（原型西安）：以古城墙为核心，最大的聚集地之一
- 故事从长安城外围的废墟开始

## 五、独特设定元素

### 源潮现象
- 仍在持续，时强时弱
- 强源潮期间，古物共鸣增强，异族也更强
- 被称为"潮汐日"

### 古物退化
- 部分古物因过度使用而"死寂"，失去共鸣能力
- 也有古物因特殊事件而"觉醒"，产生新的共鸣属性
- 造化者的能力正是赋予物品古物属性

### 信息断层
- 末世前的历史大量失传
- "古代"（即我们现在的时代）被视为失落的黄金文明
- 博物馆被视为圣地，但大多已被摧毁
- 主角的现代知识成为降维打击

### 动物觉醒
- 受源潮影响，部分动物进化出智慧
- 有善有恶，可成为伙伴也可成为敌人
- 觉醒动物同样可与古物共鸣
