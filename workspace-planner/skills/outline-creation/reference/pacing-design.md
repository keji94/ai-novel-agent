# 节奏设计手册

## 一、节奏定义

节奏 = 通过场景排列、细节密度和信息释放速度来控制故事速度和情绪强度。

---

## 二、节奏波浪理论

### 核心原则

故事节奏应该是**波浪形的**：上升→波峰→下降→再上升，且后续波峰应高于前一个。

```
情绪强度
 ↑
 |        ★大爽点         ★卷级大爽点
 |       /  \            /    \
 |      /    ★小爽点    /      \
 |     /      \/  \/  \/        ★全书级
 |    /  ★小爽点   \/  ★小爽点   \  /
 |───/───\/──────────────────────────→
 |  ★开局
 |
 +────────────────────────────────────→ 章节
```

### 波浪设计规则

1. **波峰递增**：每个单元故事的高潮应高于前一个
2. **波谷要短**：读者讨厌漫长的"无聊段"，波谷控制在1-2章
3. **波峰要够**：高潮章必须有足够的铺垫支撑，不能"草草了事"
4. **卷末是最大波峰**：每卷的结尾应该是该卷的最高点
5. **卷间可以有短暂回落**：新卷开头可以略微放缓（但不超过3章）

## 三、快节奏 vs 慢节奏

### 快节奏特征

- 动作场景、高冲突对话
- 快速视角切换
- 短句、短段落
- 直接后果
- 高信息密度
- 对话多于叙述

### 慢节奏特征

- 世界观展开、内心独白
- 关系发展、情感描写
- 细节描写、环境刻画
- 长句、复合句
- 回忆、倒叙

### 何时用快/慢

| 场景 | 推荐节奏 | 原因 |
|------|---------|------|
| 战斗/追逐 | 快 | 紧张感 |
| 对峙/谈判 | 中偏快 | 信息密度高 |
| 日常/过渡 | 慢 | 让读者喘息 |
| 世界观展开 | 中偏慢 | 需要消化 |
| 角色关系发展 | 慢→中 | 情感需要铺垫 |
| 揭秘/反转 | 快→瞬间停顿 | 冲击力 |
| 大战前夜 | 慢 | 压迫感、蓄力 |
| 大战后 | 慢→中 | 情绪释放、恢复 |

## 四、节奏调整技巧

### 4.1 加速技巧（当节奏太慢时）

- **合并场景**：把多个事件压缩到同一场景
- **删除过渡**：砍掉"路上""等待""准备"等过渡段
- **倒叙开头**：从事件的中段开始，回头补充
- **增加对话**：对话比叙述推进更快
- **缩短段落**：视觉上加快阅读速度
- **场景晚进早出**：不要从进入房间写起，从冲突开始写起；不要写到离开房间，写到冲突结束就切走

### 4.2 减速技巧（当节奏太快时）

- **增加感官细节**：视觉、听觉、触觉、嗅觉
- **展开角色内心**：主角的思考、犹豫、回忆
- **补充世界观**：通过环境描写展示设定
- **增加次要冲突**：主线之外的小摩擦
- **拉长段落**：视觉上放慢阅读速度

## 五、节奏红线

### 节奏太慢的信号

- 连续超过3章没有新的信息/事件/冲突
- 读者可以跳过整章不影响理解
- 同一个情绪/场景反复描写
- 角色在"原地踏步"（没有进展）
- 大量无关紧要的日常描写

### 节奏太快的信号

- 角色发展感觉"没铺垫就发生了"
- 读者跟不上事件发展
- 剧情有漏洞（因为跳过了必要的设定）
- 情感场景没有冲击力（因为铺垫不足）
- 读者需要反复翻回去确认

### 网文特有的节奏陷阱

- **"注水"**：为凑字数而加入无关内容（最常见的问题）
- **"赶剧情"**：为追进度而跳过必要铺垫
- **"高潮疲劳"**：连续高潮不休息，读者麻木
- **"平段过长"**：过渡段超过5章，读者流失

## 六、按平台适配

### 起点

- 允许中等节奏，可以有大段世界观/情感描写
- 每章2000-3000字，日更2-4章
- 读者容忍度较高，但仍有底线
- 适合复杂世界观和深度角色

### 番茄

- 节奏偏快，每章都需要看点
- 每章2000-2300字，日更2-6章
- "平段"不超过2章
- 适合直接爽、快节奏的故事

### 飞卢

- 极快节奏，几乎每段都有信息量
- 每章2000字，日更4-8章
- 几乎不允许"平段"
- 适合纯爽、快打快撤的故事

---

## 七、情绪节奏 vs 情节节奏

### 区分两个维度

- **情节节奏**：事件发生的速度（战斗快、日常慢）
- **情绪节奏**：读者情绪的起伏（紧张→释放→蓄力→爆发）

### 最佳实践

情节节奏和情绪节奏可以**错位**：
- **情节慢但情绪快**：大战前的宁静——情节上什么都没发生，但读者情绪高度紧张
- **情节快但情绪慢**：战后恢复——事件在推进，但读者情绪在消化

这种错位创造了丰富的阅读体验，避免"情节快=情绪快"的单调感。
