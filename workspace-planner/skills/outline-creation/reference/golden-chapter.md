# 黄金一章设计规范

## 一、核心理念

### 从黄金三章到黄金一章

传统的"黄金三章"已经不够了。当前网文市场竞争激烈，读者有数百万本书可以选择，**你只有30秒（约300字）的时间来留住读者**。

如果第一章不能让读者产生"我需要知道接下来发生什么"的冲动，就不会有第二章的阅读机会。

### 黄金一章的终极目标

> 读者读完第一章后，**必须**产生继续阅读的强烈冲动。

---

## 二、第一章结构模板

### 推荐字数：2000-3000字

```
┌──────────────────────────────────────────┐
│ 开头 300-500字：直接切入冲突/危机        │ ← 30秒钩子
│                                          │
│ 中段 1000-1500字：                       │
│   - 建立主角身份（ Who）                  │
│   - 建立核心问题（ What）                 │
│   - 建立赌注（ Why it matters）          │
│   - 金手指暗示（ Hint of power）          │
│                                          │
│ 结尾 500-800字：悬念收尾                  │ ← 追读钩子
└──────────────────────────────────────────┘
```

### 各部分详细要求

#### 2.1 开头（300-500字）：30秒钩子

**必须做到**：
- 第一句话就制造紧张感/好奇心/冲突
- 让读者立刻进入状态，没有"适应期"

**有效开头模式**：

| 模式 | 示例 | 效果 |
|------|------|------|
| **危机开头** | "苏辰醒来时，发现自己只有一条胳膊" | 瞬间紧张 |
| **冲突开头** | "你，被开除了。" | 瞬间对立 |
| **悬念开头** | "第三具尸体被发现时，所有人都在看苏辰" | 瞬间好奇 |
| **反差开头** | "全世界最强的人，正在垃圾桶里翻找食物" | 瞬间反差 |
| **动作开头** | 直接进入战斗/逃跑/追逐 | 瞬间紧张 |

**绝对禁止**：
- 大段景物描写（"夕阳西下，金色的余晖洒在破败的城墙上……"）
- 世界观说明（"在一百二十年前，世界发生了一场巨变……"）
- 大段内心独白（"苏辰想，为什么命运对他如此不公……"）
- 故弄玄虚的对话（"你知道吗……""什么？""算了，你不会想知道的。"）
- 无关紧要的日常（"苏辰像往常一样起床、洗漱、吃早饭……"）

#### 2.2 中段（1000-1500字）：建立四要素

在开头钩住读者后，迅速建立四个核心要素：

1. **Who（主角是谁）**：不是完整的人物介绍，而是通过行动展示核心特质
2. **What（核心问题）**：主角面临的直接困境/危机
3. **Why（为什么重要）**：失败的后果/赌注
4. **Hint（金手指暗示）**：暗示主角有独特之处，但不完全揭示

**关键**：通过**行动和对话**展示，而不是通过叙述说明。

#### 2.3 结尾（500-800字）：追读钩子

**必须做到**：
- 制造强烈的"然后呢？"感
- 可以是悬念、反转、危机升级、金手指即将激活

**有效结尾模式**：

| 模式 | 示例 |
|------|------|
| **悬念结尾** | "他的手指触碰到铜镜的瞬间，整个世界安静了。" |
| **危机升级** | "虫潮的声音从远处传来，越来越近。" |
| **反转结尾** | "那个一直对他微笑的老头，眼中闪过一丝不属于人类的红光。" |
| **金手指激活** | "苏辰的脑海中，一颗星辰亮了起来。" |

## 三、禁忌清单

### 第一章绝对不能做的事

1. **不能超过300字还没有冲突/危机**
2. **不能出现超过3个有名字的角色**（读者记不住）
3. **不能有大段世界观说明**（留到后续章节，通过剧情自然展开）
4. **不能有大段回忆/前史叙述**（除非直接服务于当前冲突）
5. **不能有读者删掉第一章后不影响理解后续剧情的情况**（第一章必须不可删除）
6. **不能让读者感到困惑**（可以制造悬念，但不能制造混乱）
7. **不能和后续章节的风格/节奏严重脱节**（不要用华丽的第一章骗读者进来，然后变成流水账）

---

## 四、检查清单

### 必须全部通过

- [ ] 前300字内有冲突/危机/悬念？
- [ ] 读者在30秒内能理解"发生了什么"？
- [ ] 主角的身份和处境在500字内建立？
- [ ] 核心问题/困境在1000字内建立？
- [ ] 金手指/核心优势有暗示（不需要完全揭示）？
- [ ] 有至少一个让读者"想知道接下来"的钩子？
- [ ] 没有无意义的景物描写/世界观说明/日常叙述？
- [ ] 有名字的角色不超过3个？
- [ ] 删掉第一章后，后续剧情是否受影响？（应该受影响）
- [ ] 结尾有追读钩子？

## 五、不同类型的开头适配

### 穿越/重生类
- 直接从"醒来"开始
- 通过主角的反应展示新旧世界的反差
- 金手指暗示 = 对新世界的异常熟悉/能力

### 系统类
- 系统激活作为核心钩子
- 展示系统的独特规则
- 金手指暗示 = 系统的某个特殊功能

### 灵气复苏/异能类
- 从觉醒事件开始
- 展示觉醒前后的反差
- 金手指暗示 = 觉醒结果的异常

### 都市类
- 从冲突事件开始（被开除/被甩/被追债）
- 快速建立"为什么主角需要逆袭"
- 金手指暗示 = 某个特殊机会/能力/人物的出现

### 末世/废土类
- 直接从生存危机开始
- 通过环境描写（简短）建立末世感
- 金手指暗示 = 发现异常物品/能力
