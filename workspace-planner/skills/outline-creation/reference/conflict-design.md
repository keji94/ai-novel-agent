# 冲突设计手册

## 一、冲突五步法

### 核心框架

```
对比 → 反差 → 冲突 → 转折 → 和解
```

### 1.1 对比（建立对立面）

**功能**：指出主题性的对立关系

- 强 vs 弱
- 自由 vs 压迫
- 知识 vs 无知
- 信任 vs 怀疑
- 文明 vs 野蛮

**示例**："长安城内的人享受着古物共鸣的保护，三环外的人在废墟中挣扎求生。"

### 1.2 反差（具体化差异）

**功能**：用具体场景展示对立，让读者"看到"而不是"听到"

- 从多个角度展示差异
- 重大冲突需要多重反差铺垫
- 反差越具体越有冲击力

**示例**：不只说"阶级分化"，而是写"二环的共鸣者在茶馆里讨论用哪种古物炼制首饰，三环外的拾荒者为了半瓶水拼刀子"。

### 1.3 冲突（爆发）

**功能**：对立力量正面碰撞

**冲突类型**：
- 人 vs 人（最常见）
- 人 vs 社会/体制
- 人 vs 自然/环境
- 人 vs 自我（内心冲突）
- 思想 vs 思想（价值观碰撞）

**升级原则**：
- 赌注递增：从面子到利益到自由到生命到文明
- 选择递减：从"两个都不好"到"两个都要命"
- 时间压缩：倒计时越来越短

### 1.4 转折（格局变化）

**功能**：打破现有格局，引入新的变量

- 信息揭示（隐藏真相曝光）
- 力量展示（底牌亮出）
- 外力介入（新角色/新事件改变局势）
- 角色转变（某个角色改变立场）

**关键**：转折应与主角的独特优势/金手指相关，否则主角就只是"被动的幸运儿"。

### 1.5 和解（收束）

**功能**：完成冲突弧线

**两种和解**：
- **正向和解**：转化/和解/成长（适合长期共存的对手）
- **负向和解**：消灭/驱逐/击败（适合不可调和的敌人）

**规则**：和解后不应轻易反转（除非引入更大的新冲突）。

## 二、冲突分类体系

### 按规模

| 级别 | 范围 | 示例 |
|------|------|------|
| **大矛盾** | 贯穿全书/整卷 | 人类 vs 异族、自由 vs 体制 |
| **中矛盾** | 单元故事/几章 | 与某个反派的对抗、某个任务的完成 |
| **小矛盾** | 单章/场景 | 一次争吵、一个误会 |

### 按类型

| 类型 | 说明 | 爽点潜力 |
|------|------|---------|
| **外部-人际** | 角色vs角色 | 高（直接对抗） |
| **外部-环境** | 角色vs自然/社会 | 中（可以创造紧迫感） |
| **外部-体制** | 角色vs规则/制度 | 高（读者代入感强） |
| **内部-认知** | 角色vs自己的认知局限 | 中（需要好的铺垫） |
| **内部-道德** | 角色vs自己的价值观 | 高（情感深度） |
| **内部-身份** | 角色vs自己的身份认同 | 高（代入感极强） |

## 三、冲突类型阶梯（长篇设计）

### 核心原则

300万字的长篇，冲突类型必须有**阶梯式递进**，不能在同一维度上循环。

### 推荐阶梯

```
Level 1: 物理生存（"能不能活下来"）
    ↓
Level 2: 体制压迫（"规则是不是不公正"）
    ↓
Level 3: 认知碰撞（"我以为对的不是唯一答案"）
    ↓
Level 4: 信任+身份（"我还能相信谁/我是谁"）
    ↓
Level 5: 道德两难（"没有正确答案怎么办"）
    ↓
Level 6: 终极抉择（"我选择成为什么"）
```

### 为什么是这个顺序

1. **从外到内**：先解决外部威胁，再处理内部矛盾，读者更有代入感
2. **从简单到复杂**：物理冲突最容易理解，道德两难最难——读者的理解能力随阅读深入而增长
3. **不可逆**：每一步升级后，主角无法回到"更简单"的状态（看过体制的黑暗后，无法假装不知道）

### 可选阶梯变体

根据题材不同，阶梯可以调整：

**权谋类**：
```
个人恩怨 → 家族博弈 → 朝堂争斗 → 天下大势 → 历史选择
```

**探险类**：
```
区域探索 → 大陆发现 → 秘境探险 → 位面穿越 → 世界真相
```

**都市类**：
```
个人逆袭 → 商业竞争 → 行业博弈 → 资本对抗 → 社会变革
```

## 四、反派设计原则

### 核心要求

**如果你能替反派写一篇文章论证自己是对的，这个反派就立住了。**

### 4.1 反派动机层次

好的反派至少有3层动机：

| 层次 | 说明 | 示例 |
|------|------|------|
| **表面动机** | 反派表现出来的目的 | "我要统治世界" |
| **真实动机** | 驱动反派行动的真正原因 | "我认为只有我能保护这个世界" |
| **深层动机** | 让反派成为这样的经历/价值观 | "我曾经也试图用和平方式，但失败了" |

### 4.2 反派与主角的镜像关系

最好的反派是主角的"另一种可能性"：

- 如果主角选择了A路，反派就是"选择了B路的人"
- 两人有相似的经历/起点，但做出了不同的选择
- 反派的存在让主角（和读者）思考"如果当初选了另一条路会怎样"

### 4.3 反派不需要是"人"

反派可以是：
- 一个体制/规则（比个人反派更难对抗，更有深度）
- 一种观念/信仰（思想层面的冲突）
- 主角自己（内心冲突外化）
- 自然/命运（不可抗力）

### 4.4 反派的节奏

- **不要让反派一直缺席**：终极反派在前2/3的故事中必须有存在感（即使是间接的）
- **不要让反派一次性出场**：反派的力量/威胁应该逐步揭示
- **不要让反派突然洗白**：转变需要足够的铺垫和触发事件
