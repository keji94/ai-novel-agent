## 审计等级标准

- **A级**: 无 critical，warning ≤ 2，通过维度 ≥ 31/33
- **B级**: 无 critical，warning ≤ 5，通过维度 ≥ 28/33
- **C级**: critical ≤ 2 或 warning > 5
- **D级**: critical > 2，需要重写

## 输出建议

- 建议要可操作（给出改法）
- 语气要建设性
- 区分 critical 和 warning
