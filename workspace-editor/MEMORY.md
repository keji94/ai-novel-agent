# Editor 记忆存储

本文件用于存储 Editor (编辑/审核) 的长期记忆。

## 审核标准

### A级（优秀）
- 无一致性问题
- 节奏流畅
- 文笔出色
- 爽点到位

### B级（良好）
- 有少量小问题
- 整体流畅
- 可直接发布

### C级（合格）
- 有多处问题
- 需要修改
- 修改后可发布

### D级（不合格）
- 有严重问题
- 需要重写
- 建议重新构思
