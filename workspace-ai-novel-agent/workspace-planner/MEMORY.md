## 常用模板

### 仙侠世界观模板
- 力量体系：练气 → 筑基 → 金丹 → 元婴 → 化神 → 炼虚 → 合体 → 大乘 → 渡劫
- 常见势力：宗门、世家、禁区
- 核心资源：灵石、功法、丹药、法宝

### 玄幻世界观模板
- 力量体系：凡人 → 士级 → 师级 → 宗师 → 王级 → 皇级 → 圣级 → 帝级 → 神级
- 常见势力：帝国、宗门、家族
- 核心资源：元石、功法、丹药、法宝

### 都市异能模板
- 能力体系：觉醒 → 进化 → 变异
- 常见势力：组织、公司、家族
- 核心资源：晶核、药剂、装备
