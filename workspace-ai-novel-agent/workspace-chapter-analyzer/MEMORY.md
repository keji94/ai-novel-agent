# ChapterAnalyzer 记忆存储

本文件用于存储 ChapterAnalyzer (章节分析器) 的长期记忆。

## 常见角色提取模式

### 姓名格式
- 中文: 2-4字，首字大写
- 外文: 英文名/音译名

### 称呼格式
- 关系称呼: 师兄/师姐/师弟/师妹/师父
- 职位称呼: 长老/掌门/宗主
- 尊称: 前辈/阁下/道友

## 常见资源变化模式

### 获得模式
- "获得/得到/取出/收到/捡到/抢到"
- "炼制/炼化/吸收"

### 消耗模式
- "用掉/消耗/失去/献祭/丢弃"
- "炼化(消耗原材料)"

## 伏笔识别信号词

### 埋设信号
- 心中隐隐觉得
- 似乎有些不对
- 却没注意到
- 并未察觉
- 不知为何
- 若有所思

### 回收信号
- 原来
- 难怪
- 终于明白
- 真相大白
- 原来如此
- 一切都说得通了

## 情绪词汇表

### 正面情绪
- 喜悦: 喜、乐、高兴、开心、愉悦
- 期待: 期待、盼望、希望、渴望
- 感激: 感激、感谢、感恩
- 自信: 自信、笃定、从容

### 负面情绪
- 愤怒: 怒、气、愤、火、暴怒
- 悲伤: 悲、伤、痛、哭、泪
- 恐惧: 惧、怕、恐、惊、惶恐
- 焦虑: 焦虑、担忧、不安

### 中性情绪
- 惊讶: 惊、愣、怔、呆、震惊
- 疑惑: 疑惑、不解、困惑
- 沉思: 沉思、思考、琢磨

## 注意事项

- 优先提取明确出现的信息
- 对推断信息标注"待确认"
- 发现矛盾时记录并报告
- 保留信息来源（章节号）
