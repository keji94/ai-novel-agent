# 特殊修订能力示例

## 信息越界修复

当检测到角色知道不该知道的信息时：

**修复策略**:
1. 删除角色知晓的描写
2. 改为角色猜测/怀疑
3. 或增加角色获知信息的合理途径

```
原文:
林风心中了然，苏婉一定是去了后山秘地。

修复后:
林风心中疑惑，苏婉行踪神秘，莫非是去了后山？他摇了摇头，自己一个外门弟子，又怎能知道内门的秘密。
```

## 配角降智修复

```
问题: 赵天作为筑基期高手，竟然被练气期的林风一句话骗了。

修复策略:
1. 增加被骗的合理原因
2. 或修改为看穿但另有打算

原文:
赵天哈哈大笑："好！我答应你！"

修复后:
赵天眯起眼睛，筑基期的神识扫过林风，确认对方确实只有练气期修为。他心中冷笑，一个练气期小辈，就算有阴谋又能如何？于是他开口道："好！我答应你！"但他的神识始终锁定着林风，随时准备出手。
```

## 战力崩坏修复

```
问题: 林风练气期一剑击败筑基期敌人。

修复策略:
1. 增加敌人轻敌/受伤/被偷袭的原因
2. 或修改为林风险胜
3. 或降低林风表现，改为逃跑

修复后:
林风借着敌人轻敌的一瞬，全力催动灵力，玄铁剑上灵光大盛。但他心中清楚，这只是拼死一击，若不能重创对方，自己必死无疑。剑光闪过，对方只是受了轻伤，而林风已灵力耗尽，踉跄后退。
```
