# AI 章节解析流程

## 概述

在获取到每本书的章节内容后，使用AI对章节进行深度分析，产出结构化的书籍分析报告（Markdown格式）。

---

## 解析目标

| 分析项 | 数据来源 | 说明 |
|-------|---------|------|
| 书名 | 接口返回 | 书籍名称 |
| 分类 | 接口返回 | 玄幻/都市/仙侠等 |
| 作者 | 接口返回 | 作者名称 |
| 作者等级 | 接口返回 | lv1-lv5 |
| 作品简介 | 接口返回 | 书籍简介 |
| 金手指 | AI分析 | 主角的特殊能力/系统/外挂 |
| 世界观 | AI分析 | 故事的世界设定和力量体系 |
| 前三章剧情梗概 | AI分析 | 开篇核心情节 |
| 前二十章剧情梗概 | AI分析 | 故事循环套路总结 |

---

## 处理策略

由于章节内容较长，采用**分层解析**策略：

```
第零层: 前3章精读 → 提取世界观设定
第一层: 前三章精读 → 生成开篇梗概 + 提取金手指
第二层: 前20章速读 → 总结故事套路和节奏
第三层: 前50章浏览 → 把握整体剧情走向（可选）
```

## Step 5.0: 世界观解析

**输入**: 前3章内容

**AI Prompt**:
```
你是一位资深的网文分析师，请分析以下小说章节，提取故事的世界观设定。

## 书籍信息
- 书名: {bookName}
- 作者: {authorName}
- 分类: {category}

## 章节内容
{chapters_1_to_3}

## 任务
请识别本书的世界观设定，回答以下问题：

1. **世界类型**（选择最贴切的一个或组合）：
   - 修仙世界：以修炼成仙为目标，有灵气、丹药、法宝等设定
   - 玄幻世界：异能/斗气/魔法等超自然力量体系
   - 都市异能：现代都市背景下的超能力/异能设定
   - 科幻未来：高科技/星际/机甲/赛博朋克等
   - 历史架空：基于历史背景的虚构世界
   - 灵异悬疑：鬼怪/阴阳/风水/玄学等设定
   - 游戏/虚拟：VR游戏/穿越到游戏世界
   - 奇幻西幻：西方奇幻风格，魔法/龙/精灵等
   - 武侠江湖：传统武侠，内功/轻功/门派等
   - 末世废土：丧尸/灾难后的生存世界
   - 其他：请具体描述

2. **核心设定**（100字以内）：
   描述这个世界最核心的设定规则，包括：
   - 世界的基本运行逻辑
   - 独特的社会结构或组织形式
   - 关键的地理/空间设定（如有）

3. **力量体系**（80字以内）：
   - 这个世界的力量如何划分等级/层次？
   - 主角处于什么位置？目标是什么？
   - 晋升/升级的基本路径是什么？

请以JSON格式输出：
{
  "worldType": "世界类型",
  "coreSetting": "核心设定描述",
  "powerSystem": "力量体系描述"
}
```

**输出示例**:
```json
{
  "worldType": "修仙世界",
  "coreSetting": "一个弱肉强食的修仙世界，各大宗门林立，散修生存艰难。修仙者需吸纳天地灵气，炼制丹药法宝，追求长生大道。世俗皇权在修仙者面前不值一提。",
  "powerSystem": "练气→筑基→结丹→元婴→化神→炼虚→合体→大乘→渡劫。主角从凡人起步，目标是通过修炼不断提升境界，最终飞升成仙。"
}
```

## Step 5.1: 金手指提取

**输入**: 前3章内容

**AI Prompt**:
```
你是一位资深的网文分析师，请分析以下小说章节，提取主角的"金手指"。

## 书籍信息
- 书名: {bookName}
- 作者: {authorName}
- 分类: {category}

## 章节内容
{chapters_1_to_3}

## 任务
请识别主角的"金手指"（特殊能力/系统/外挂/独特优势）：

1. **金手指类型**（选择最贴切的一个）：
   - 系统流：获得神秘系统/面板/APP
   - 重生流：带着前世记忆重生
   - 穿越流：穿越到异世界/历史
   - 空间流：拥有随身空间/小世界
   - 天赋流：天生异能/特殊体质
   - 传承流：获得大佬传承/记忆
   - 物品流：拥有特殊法宝/神器
   - 其他：请具体描述
   - 暂无明显金手指

2. **金手指描述**（50字以内）：
   用简洁的语言描述金手指的具体功能和限制

3. **金手指展现方式**（30字以内）：
   金手指是如何在开篇呈现的

请以JSON格式输出：
{
  "type": "金手指类型",
  "description": "具体描述",
  "presentation": "展现方式"
}
```

## Step 5.4: 前20章剧情梗概

**输入**: 前20章内容（如果章节不足20章，则取全部章节）

**AI Prompt**:
```
你是一位资深的网文分析师，请为以下小说撰写前20章的**整体剧情脉络**。

## 书籍信息
- 书名: {bookName}
- 作者: {authorName}
- 分类: {category}

## 章节内容（前20章）
{chapters_1_to_20}

## ⚠️ 重要要求
1. **不要重复前三章的细节**！前三章已有单独梗概，这里聚焦于整体发展脉络
2. **宏观视角**：聚焦于整体剧情走向、势力发展、关键转折，而非逐章复述
3. **核心内容**：
   - 第4-20章的主要剧情线和阶段变化
   - 关键转折点与高潮事件
   - 势力/能力的发展轨迹
   - 核心伏笔与悬念
4. 用300-500字概括，语言精炼有力

## 输出格式
直接输出梗概文本，分段呈现，不要添加标题。
```

## 报告输出示例

```markdown
# 📖 【凡人修仙传】
仙侠 | 修仙 | 凡人流 | 忘语 lv5
---

## 📝 作品简介
一个普通山村少年，偶然之下进入当地江湖小门派，成为一名记名弟子。他以这样的身份，如何在这竞争激烈、弱肉强食的修仙世界中生存下来，并最终走上修仙之路？

---

## 🌍 世界观：【修仙世界】
一个弱肉强食的修仙世界，各大宗门林立，散修生存艰难。修仙者需吸纳天地灵气，炼制丹药法宝，追求长生大道。世俗皇权在修仙者面前不值一提。力量体系为练气→筑基→结丹→元婴→化神→炼虚→合体→大乘→渡劫，最终飞升成仙。

---

## ✨ 金手指：【小绿瓶】
神秘小绿瓶，可催熟灵药，极大降低修炼资源门槛。这个金手指功能单一但实用，让资质平平的主角能够暗中积累资源，逐步缩小与天才们的差距。金手指的展现方式是开篇捡到小绿瓶，偶然发现其催熟能力，开始暗中收集种子进行实验。

---

## 🔥 黄金一章剧情梗概
**章节：第1章《山村少年》**
山村少年韩立家境贫寒，被父母送入七玄门学艺。入门测试中，他资质平平被分配到边缘的记名弟子名额。神秘的墨大夫看中韩立，收其为徒教授医术。韩立在山中发现一个神秘的小绿瓶，并在无意中发现这瓶子能够催熟植物——这是全书最重要的金手指首次登场，也是凡人流"资质不够，资源来凑"的核心设定初现。

---

## 📚 前三章剧情梗概
- **第1章**：韩立被送入七玄门，资质平平成为记名弟子，被神秘墨大夫收徒，意外获得小绿瓶
- **第2章**：韩立跟随墨大夫学习医术，暗中实验小绿瓶催熟功能，发现其能催熟灵药
- **第3章**：韩立开始秘密收集种子，利用小绿瓶催熟灵药暗中积累资源，同时发现墨大夫似乎另有图谋

---

## 📖 前20章剧情梗概
韩立在墨大夫指导下学习医术和基础武功，同时利用小绿瓶催熟灵药暗中积累修炼资源。随着修炼深入，他发现墨大夫的真实目的——夺舍自己。经过一番生死搏斗，韩立反杀墨大夫，获得了长春功等修仙功法。

他正式踏上修仙之路，在后续冒险中不断提升实力。从七玄门到修仙界，韩立始终保持低调谨慎，利用小绿瓶的资源优势，在关键时刻反败为胜。从一个凡人少年逐渐成长为练气期修士，建立了"扮猪吃虎"的行事风格。

核心伏笔：小绿瓶的真正来历、修仙界的更大阴谋、韩立的修仙之路能走多远。
```
