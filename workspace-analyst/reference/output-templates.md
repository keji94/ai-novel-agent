# 输出格式模板

## 作品分析报告

```markdown
# 《{作品名}》分析报告

## 基本信息
- 作者: {作者}
- 类型: {类型}
- 字数: {字数}
- 平台: {平台}

## 结构分析

### 开篇分析
[开篇设计、钩子、节奏]

### 发展分析
[情节推进、冲突设计、节奏控制]

### 高潮分析
[高潮设计、爽点密度、情感爆发]

### 结尾分析
[结尾设计、收束方式、余韵]

## 设定分析

### 世界观
[世界观设计、独特性、扩展性]

### 力量体系
[体系设计、平衡性、成长空间]

### 角色设计
[主要角色、配角功能、反派设计]

## 爽点分析

### 爽点类型
- 装逼打脸: {数量}次
- 逆袭成长: {数量}次
- 智斗布局: {数量}次
- 热血战斗: {数量}次

### 爽点设计手法
[分析具体手法]

## 节奏分析

### 章节节奏
[每章字数、节奏分布]

### 卷节奏
[每卷结构、节奏曲线]

### 整体节奏
[全书节奏、高潮分布]

## 风格分析

### 叙事风格
[叙事视角、叙事方式]

### 文字风格
[文字特点、修辞使用]

### 对话风格
[对话特点、口语化程度]

## 成功要素总结

1. [要素1]
2. [要素2]
3. [要素3]

## 可复用技巧

1. [技巧1]
2. [技巧2]
3. [技巧3]
```

---

## 技巧提取报告

```markdown
# 技巧提取报告

## 来源信息
- 作品: {作品名}
- 平台: {平台}
- 章节: {章节}

## 提取技巧

### 技巧1: {技巧名}
- 类型: 开篇/节奏/结尾/描写/对话
- 描述: {技巧描述}
- 示例: {原文示例}
- 应用场景: {适用场景}

### 技巧2: {技巧名}
...
```

```json
{
  "status": "success",
  "response": "已完成《{作品名}》分析，提取了{N}个可复用技巧",
  "analysis_report": "./knowledge/analysis/{作品名}_分析报告.md",
  "techniques_extracted": 15,
  "techniques": [
    {
      "name": "开篇悬念钩子",
      "category": "structure"
    }
  ]
}
```
