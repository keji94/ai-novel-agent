# 情绪动态与拉扯设计

## 技巧描述
情绪点是触发读者情绪的起点（如遇到恶人→愤怒），情绪动态则是情绪随剧情发展的转变过程（如愤怒→兴奋），情绪拉扯通过友方付出或敌方迫害事件推动主角产生主动或被动的欲望目标。

## 核心要点
1. **情绪点**：让读者开始产生情绪的触发点，是产生欲望目标的催动剂（遇到恶人→愤怒，生死→惧怕，暧昧→开心）
2. **情绪动态**：随剧情发展情绪状态发生转变（愤怒→兴奋，恐惧→忧虑，开心→愉快/难过）
3. **情绪拉扯和推动**：
   - 友方付出事件 → 主角产生主动欲望目标（报恩、守护）
   - 敌方迫害事件 → 主角产生被动欲望目标（复仇、自保）
4. **核心公式**：情绪点触发 → 情绪动态转变 → 产生欲望目标 → 驱动情节推进

## 应用场景
- 设计每章的情绪转折点
- 构建主角的内在动机链
- 设计配角对主角的情绪影响
- 制造情节驱动力

## 示例
> 主角看到师父被反派围攻（情绪点：愤怒+恐惧）→ 师父为保护主角身受重伤（情绪动态：愤怒→悲痛+愧疚）→ 主角发誓变强复仇（拉扯推动→产生主动欲望目标）→ 开始修炼之旅（情节推进）

## 注意事项
- 情绪转变要有逻辑，不能跳跃式变化
- 情绪拉扯不能过度使用，读者会疲劳
- 欲望目标必须与情绪直接关联，否则显得刻意
