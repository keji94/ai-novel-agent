# 三大平台合规检查体系

## 技巧描述
系统化的网文平台合规检查方法，覆盖通用违规红线和三大平台（番茄、起点、纵横）的专属审核要求，包含敏感词筛查的谐音/变体识别。

## 核心要点
1. **通用违规六大类**：低俗色情、暴力血腥、封建迷信、时政敏感、民族歧视、宗教敏感
2. **番茄专属**：标题党、虚假宣传、诱导互动、低俗擦边人设
3. **起点专属**：版权问题、内容同质化、违规导流、虚假人设、分类规范
4. **纵横专属**：情节拖沓、无核心看点、违规广告植入
5. **敏感词筛查**：不仅检查原词，还需覆盖谐音、变体、拆字、拼音替代等形式

## 应用场景
- 新书投稿前的合规自查
- 编辑审核稿件时的第一道检查
- 批量更新章节时的合规复核
- 跨平台分发时的适配性审查

## 示例

**通用违规检查清单**：
- 是否涉及封建迷信？→ 检查"修炼""求神拜佛"等描写是否过度
- 是否涉及时政敏感？→ 检查是否有影射现实的情节
- 是否涉及民族歧视？→ 检查对特定民族的描写是否负面刻板

**敏感词变体示例**：
- 原词 → 谐音/拆字/拼音 → 均需排查
- 如"自杀" → "紫砂"/"zi sha" → 同样违规

## 注意事项
- 平台审核标准会随政策调整，需定期更新检查清单
- 不同平台的红线标准有差异，跨平台发布需分别适配
- 敏感词筛查建议结合自动化工具+人工复核
