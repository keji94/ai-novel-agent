# 竞争力差异化检查

## 技巧描述
从差异化亮点、读者留存和平台适配三个维度检查作品竞争力，确保新书在同质化严重的市场中脱颖而出。

## 核心要点
1. **差异化亮点检查**：独特世界观 / 独特人设 / 独特剧情设定，至少占一项
2. **读者留存检查**：每章结尾钩子是否有效，读者是否有继续阅读的动力
3. **平台适配性优化**：根据目标平台的读者画像调整风格和节奏

## 应用场景
- 新书大纲评审
- 编辑初审时的竞争力评估
- 已有作品的市场定位诊断

## 示例

**差异化检查清单**：
- 世界观：是否有不同于同类作品的独特设定？
  - ✗ 标准修仙升级流 → ❌ 无差异化
  - ✓ 修仙+赛博朋克 → ✅ 有差异化
- 人设：主角是否有记忆点？
  - ✗ 常见废柴逆袭 → ❌ 无差异化
  - ✓ 盲眼剑客修仙 → ✅ 有差异化
- 剧情设定：是否有独特的核心驱动？
  - ✗ 打怪升级复仇 → ❌ 无差异化
  - ✓ 倒计时30天拯救世界 → ✅ 有差异化

## 注意事项
- 差异化不等于猎奇，需服务于故事核心
- 钩子有效性需站在读者角度判断，而非作者自嗨
- 平台适配不是迎合，而是找到作品与平台读者的最佳契合点
