# character - 角色塑造 分类索引

## 技巧列表

### T001 默默付出型女主模板
- 评分: B | 难度: beginner
- 一句话: 设定长相出众、默默付出的柔弱型女角色
- 场景: 男频网文核心女主/女配，营造被守护感
- 标签: 女主, 人设, 柔弱, 默默付出, 套路化

### T002 标签化与脸谱化
- 评分: A | 难度: beginner
- 一句话: 给角色绑定1-2个强辨识度标签，让读者快速记忆
- 场景: 长篇小说配角众多的场景，角色首次登场印象建立
- 标签: 标签化, 脸谱化, 记忆锚点, 角色辨识度

### T003 现实人物化用
- 评分: A | 难度: intermediate
- 一句话: 将生活中或影视动漫中的角色特质移植到小说角色上
- 场景: 需要快速建立角色立体感，增强读者代入感
- 标签: 现实人物, 化用, 代入感, 经典角色

### T004 言行一致原则
- 评分: A | 难度: beginner
- 一句话: 角色言行必须符合人设定位和大众逻辑常识（必修底线）
- 场景: 所有角色创作场景，角色关键行为决策
- 标签: 言行一致, 人设底线, 逻辑常识, 新人常见错误

### T017 人物成长轨迹设计
- 评分: A | 难度: intermediate
- 一句话: 成长型主角变化需逐步推进，反派动机需呈现，人物互动需自然
- 场景: 主角成长线规划，反派深度塑造，人物关系发展
- 标签: 成长, 人物弧光, 反派, 互动, 人设递进
