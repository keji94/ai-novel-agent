# 角色与主题检查

## 一、角色弧线验证

### Want vs Need 分析

对每个**核心角色**（主角 + 至少 3 个配角）：

```
角色：__________
Want（表面欲望）：__________
Need（深层需求）：__________

Want 和 Need 是否不同？ 是/否
转变触发事件：__________
弧线类型：正面（成长）/ 平面（不变但有主题作用）/ 负面（堕落）
```

| 弧线类型 | 描述 | 适合角色 |
|---------|------|---------|
| 正面弧 | Want→Need 转变，角色成长 | 主角、核心盟友 |
| 平面弧 | 角色不变，但改变周围人 | 导师型角色 |
| 负面弧 | Need→Want 堕落，角色衰败 | 反派、悲剧角色 |
| 弹性弧 | 反复在 Want 和 Need 间摇摆 | 复杂角色 |

### 常见问题

| 问题 | 严重度 | 描述 |
|------|--------|------|
| 主角无弧线 | P0 | 主角从头到尾没有变化 |
| Want=Need | P1 | 没有成长空间，弧线是平的 |
| 转变无触发 | P2 | 角色突然变了，缺乏触发事件 |
| 核心配角无弧线 | P2 | 配角全程工具人 |
| 弧线未完成 | P1 | 角色开始了转变但结局前没完成 |

## 二、关系网张力图

### 检查方法

对每对**核心角色关系**，逐卷标注状态：

```
角色对：苏辰 ↔ 沈墨寒
V1: 漠视 → V2: 警惕 → V3: （无交集）→ V4: 打压 → V5: 不情愿盟友 → V6: 被迫合作

状态是否有变化？ 是 ✓
变化是否有因果关系？ 是 ✓
```

### 关系张力标准

| 评估 | 标准 |
|------|------|
| 优秀 | 每对关系至少有 2 次状态变化 |
| 合格 | 每对关系至少有 1 次状态变化 |
| 问题 | 某对关系全程状态不变 |
| 严重 | 重要关系（如主角与核心对手）全程无交集 |

### 常见问题

| 问题 | 严重度 | 描述 |
|------|--------|------|
| 关系冻结 | P2 | 核心关系全程无变化 |
| 无因果关系 | P2 | 关系变了但没有原因 |
| 过于简单 | P3 | 关系只有"敌/友"两种状态 |
| 遗忘关系 | P1 | 某卷中重要角色完全消失 |

## 三、反派合理性检查

### 检查清单

| # | 检查项 | 合格标准 |
|---|--------|---------|
| 1 | 动机自洽 | 反派的行为是否可以用他自己的逻辑解释？ |
| 2 | 威胁递进 | 反派的威胁是否随故事推进而升级？ |
| 3 | 对抗升级 | 主角与反派的对抗是否有明确的升级弧线？ |
| 4 | 灰度 | 反派是否有"说得通"的一面？（非纯恶） |
| 5 | 独立意志 | 反派是否在自己的故事里是"主角"？ |
| 6 | 失败逻辑 | 反派最终失败是否因为自己的选择/缺陷，而非主角开挂？ |

### 威胁递进检查

```
V1: 反派威胁程度 ★
V2: ★★
V3: ★★★
...
每卷威胁是否递增？ 是/否
```

### 常见问题

| 问题 | 严重度 | 描述 |
|------|--------|------|
| 反派无动机 | P1 | 反派作恶无理由 |
| 威胁不递进 | P2 | 反派从头到尾同样威胁级别 |
| 反派降智 | P1 | 反派在关键时刻变蠢，给主角送胜利 |
| 多反派混乱 | P2 | 反派太多没有主次，读者记不住 |

## 四、主题一致性检查

### 检查方法

1. 提取核心主题（通常一句话）
2. 检查每卷的主要冲突是否在服务这个主题
3. 检查主角的成长是否围绕这个主题

### 检查模板

```
核心主题：__________（如"制度vs个人"）

| 卷 | 主要冲突 | 是否服务主题 | 关联方式 |
|----|---------|------------|---------|
| V1 | 生存 | 是/否 | ... |
| V2 | ... | ... | ... |

主题在各卷的呈现方式是否多样？（不能每卷都是同一种"制度vs个人"）
主角的成长弧线是否最终回答了主题的问题？
```

### 常见问题

| 问题 | 严重度 | 描述 |
|------|--------|------|
| 无主题 | P1 | 各卷各自为战，没有一个统一的主题 |
| 主题说教 | P2 | 主题通过角色嘴巴直接说出，而非通过剧情体现 |
| 主题不一致 | P2 | 某些卷的冲突与核心主题无关 |
| 主题无回答 | P1 | 提出了主题问题但结局没有回答 |
