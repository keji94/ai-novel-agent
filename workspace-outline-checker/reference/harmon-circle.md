# Dan Harmon Story Circle 检查

> 来源：Dan Harmon 的故事圈理论（Joseph Campbell 英雄之旅的简化版）

## 8 步检查

```
        1.YOU(舒适区)
       ↗              ↘
  8.CHANGE              2.NEED
  (改变)               (需要)
     ↑                    ↓
  7.RETURN            3.GO
  (回归)               (出发)
       ↖              ↙
   6.TAKE  ←  5.FIND ←  4.SEARCH
  (付出代价)  (找到)     (寻找)
```

| # | 步骤 | 英文 | 检查要点 | 常见问题 |
|---|------|------|---------|---------|
| 1 | 你（舒适区） | YOU | 开篇主角是否处于"舒适但不满足"的状态？是否有明确的缺陷（Want）和深层需求（Need）？ | 开篇主角太完美无缺 / 没有展示"不满足" |
| 2 | 需要 | NEED | 什么打破了舒适？主角的欲望是否被明确唤醒？这个 Need 是否和真正的 Need（主题）不同？ | Want 和 Need 相同 → 角色没有成长空间 |
| 3 | 出发 | GO | 主角是否**主动**踏入未知？还是被被动推着走？主动选择比被动推入更有力 | 主角全程被动，没有主动决策时刻 |
| 4 | 寻找 | SEARCH | 主角在新世界中学习、适应、试错。是否有"获得工具/盟友/知识"的过程？ | 主角一下就什么都会，缺乏学习过程 |
| 5 | 找到 | FIND | 主角得到了他 **Want** 的东西（表面欲望）。但这是 False Victory 吗？ | 没有"找到"的满足感 → 前半段缺乏回报 |
| 6 | 付出代价 | TAKE | 主角为得到的东西**付出了什么代价**？是否有失去？这个代价是否让他重新审视 Want vs Need？ | 没有代价 → 故事缺乏重量感 |
| 7 | 回归 | RETURN | 主角带着新认知回归（不一定是物理回归）。是否将 Need 融入了新自我？ | 回归后和出发前没有变化 |
| 8 | 改变 | CHANGE | 结局主角是否**本质上不同**了？Opening Image vs Final Image 是否形成反差？ | 主角从头到尾没变 / 变化突兀缺乏铺垫 |

## 多卷映射

| 步骤 | 全书位置 | 检查方法 |
|------|---------|---------|
| 1. YOU | V1 开篇 | 主角初始状态、缺陷、不满足 |
| 2. NEED | V1 催化事件 | 欲望被唤醒的触发事件 |
| 3. GO | V1-V2 跨越门槛 | 主动踏入更大的世界 |
| 4. SEARCH | V2-V4 | 学习、试错、积累的过程 |
| 5. FIND | V4-V5 | 获得表面欲望（权力/地位/真相） |
| 6. TAKE | V4 末-V5 初 | 付出代价（被流放/失去信任/自我怀疑） |
| 7. RETURN | V5-V6 | 带着新认知回归 |
| 8. CHANGE | V6 结局 | 本质性的改变 |

## 角色内部弧线检查

对每个**核心角色**（不只是主角），用 Story Circle 单独检查：

```
角色 Want: ______________（表面欲望）
角色 Need: ______________（深层需求，通常角色自己不知道）

8步是否齐全？哪步缺失？
Want 和 Need 是否不同？（相同=没成长空间）
Want→Need 的转变是否有明确的触发事件？
```

## 常见问题清单

| 问题 | 严重度 | 描述 |
|------|--------|------|
| 主角 Want=Need | P1 | 没有成长空间，弧线是平的 |
| 无"主动出发"时刻 | P1 | 主角全程被剧情推着走 |
| 无代价 | P0 | 故事没有重量感，获得一切无失去 |
| SEARCH 阶段无学习 | P2 | 主角没有成长过程，一出场就够强 |
| 无 CHANGE | P0 | 开头和结尾主角状态相同 |
| 核心配角无弧线 | P2 | 配角功能化，没有自己的 Want→Need |
| 代价与 Want 不对等 | P3 | 付出太少或代价与主题无关 |
