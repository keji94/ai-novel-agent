# Snyder 15-beat 结构检查

> 来源：Blake Snyder《Save the Cat》

## 15 个节拍

| # | 节拍 | 位置（全书） | 位置（单卷） | 检查要点 |
|---|------|------------|------------|---------|
| 1 | Opening Image | 0-1% | 第1章 | 开篇画面：主角的"之前"状态是否清晰？是否与 Final Image 形成反差？ |
| 2 | Theme Stated | 5% | ch 1-2 | 主题暗示：有人（非主角）说了全书主题，主角当时没懂。是否存在？ |
| 3 | Set-Up | 1-10% | ch 1-5 | 铺垫：主角的缺陷、需要、环境是否全部展示？六要素（主角、缺陷、需要、盟友、对手、赌注）是否齐全？ |
| 4 | Catalyst | 10% | ch 5-8 | 催化事件：打破现状的事件是否有力？主角是否无法忽视？ |
| 5 | Debate | 10-20% | ch 8-15 | 犹豫：主角是否经历了"要不要踏入未知"的内心挣扎？ |
| 6 | Break into Two | 20% | ch 15-18 | 跨越门槛：主角主动决定进入新世界/新阶段。是否是主动选择？ |
| 7 | B Story | 22% | ch 18-20 | B故事线：承载主题的关系线是否建立？（通常是感情线或导师线） |
| 8 | Fun and Games | 20-50% | ch 18-35 | 娱乐与游戏：主角在新世界的"试炼"——这是爽点最集中的区域。爽点密度是否足够？ |
| 9 | Midpoint | 50% | ch 25-28 | 中点：False Victory（虚假胜利+隐患）或 False Defeat（看似失败实则转机）。是否有明确转折？赌注是否提升？ |
| 10 | Bad Guys Close In | 50-75% | ch 28-40 | 反击收紧：外部压力增大+内部出现裂痕。是否同时有外压和内裂？ |
| 11 | All Is Lost | 75% | ch 35-40 | 全部失去：最低谷时刻。是否有"死亡气息"（隐喻的或字面的）？是否失去了什么重要的？ |
| 12 | Dark Night of Soul | 75-80% | ch 38-42 | 灵魂暗夜：主角消化失去、找到新力量的过程。是否有内心转折？ |
| 13 | Break into Three | 80% | ch 40-42 | 跨入第三幕：A故事（外部冲突）+ B故事（主题线）+ 主角新认知三者汇合。是否同时解决？ |
| 14 | Finale | 80-99% | ch 42-50 | 高潮：主角用新认知+新能力解决最终冲突。是否有"风暴城堡"→"执行计划"→"高塔揭秘"→"挖掘深处"的递进？ |
| 15 | Final Image | 99-100% | 末章 | 终章画面：与 Opening Image 形成反差。主角是否"变了"？ |

## 多卷映射

全书 N 卷时，按以下比例映射：

| 节拍 | 全书位置 | 典型所在卷 |
|------|---------|-----------|
| Opening Image | V1 第1章 | V1 |
| Theme Stated | V1 前5章 | V1 |
| Set-Up | V1 全卷 | V1 |
| Catalyst | V1 中后段 | V1 |
| Debate | V1 末 ~ V2 初 | V1-V2 |
| Break into Two | V2 前半 | V2 |
| B Story | V2 中段 | V2 |
| Fun and Games | V2-V3 | V2-V3 |
| **Midpoint** | **V3 末 ~ V4 初** | **V3-V4** |
| Bad Guys Close In | V4 | V4 |
| All Is Lost | V4 末 ~ V5 初 | V4-V5 |
| Dark Night of Soul | V5 前半 | V5 |
| Break into Three | V5 后半 | V5 |
| Finale | V6 | V6 |
| Final Image | V6 末章 | V6 |

## 常见问题

| 问题 | 严重度 | 描述 |
|------|--------|------|
| 无 Midpoint | P0 | 全书中点没有明确转折，前后半差异模糊 |
| Midpoint 不锐利 | P1 | 有转折但不够sharp，从上升到下降缺乏断崖感 |
| B Story 缺失 | P1 | 没有独立于主线的关系线承载主题 |
| Fun and Games 干旱 | P1 | 最应该有爽点的区域却没有爽点 |
| All Is Lost 不够低 | P1 | 低谷不够深，后续回升缺乏力度 |
| Opening/Final Image 无反差 | P2 | 开头和结尾主角状态变化不明显 |
| Theme Stated 缺失 | P2 | 主题没有被明确暗示 |
| Debate 太短 | P3 | 主角太快接受新世界，缺乏犹豫 |
