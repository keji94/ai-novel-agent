# 网文节奏检查

> 网文特有维度，直接影响追读和付费。

## 一、爽点密度分析

### 爽点类型

| 类型 | 描述 | 示例 |
|------|------|------|
| 知识碾压 | 主角用知识/认知碾压对手 | 识破假古物 |
| 战斗碾压 | 主角实力碾压 | 一招击败 |
| 打脸 | 之前看不起主角的人被震惊 | 天赋测试全场震惊 |
| 制度爽 | 用规则/制度解决问题 | 用数据对抗打压 |
| 情感释放 | 长期压抑后的爆发 | 暴走击杀虫将 |
| 实力突破 | 升级/觉醒 | 四阶突破 |
| 真相揭示 | 悬念揭开 | 发现母亲日志 |
| 身份反转 | 身份/地位反转 | 盟主回归 |
| 智谋博弈 | 策略胜利 | 盐荒事件局部胜 |

### 密度标准

| 级别 | 密度 | 评估 |
|------|------|------|
| 优秀 | 每 3-5 章一个爽点 | 追读体验极好 |
| 合格 | 每 6-8 章一个爽点 | 节奏可控 |
| 警告 | 9-14 章无爽点 | 可能出现弃读 |
| 危险 | 15 章以上无爽点 | **P2 问题**，几乎必然弃读 |

### 检查方法

1. 逐章标注爽点类型（或标注"无"）
2. 计算连续无爽点章节数
3. 标记"干旱区"（15章+无爽点）
4. 检查爽点类型是否多样（不要连续同类型）

## 二、章节钩子检查

### 钩子类型

| 类型 | 描述 | 力度 |
|------|------|------|
| 悬念钩 | 抛出新问题 | ★★★★★ |
| 危机钩 | 主角陷入危险 | ★★★★ |
| 反转钩 | 预期被打破 | ★★★★ |
| 期待钩 | 预告即将发生的事 | ★★★ |
| 情感钩 | 情感未释放 | ★★★ |
| 信息钩 | 新信息引发好奇 | ★★ |

### 检查方法

逐章检查末尾是否设计了"不得不看下一章"的驱动力。标记无钩子的章节。

## 三、断点预测

### 高风险弃读模式

| 模式 | 描述 | 风险等级 |
|------|------|---------|
| **连续低谷** | 3章+情绪持续低沉无释放 | ★★★★★ |
| **信息过载** | 连续大量世界观设定/背景介绍 | ★★★★ |
| **无冲突段落** | 5章+无明确冲突推动 | ★★★★ |
| **重复模式** | 连续3次+相似套路（打脸→打脸→打脸） | ★★★ |
| **视角跳离** | 主角视角长时间缺席 | ★★★ |
| **悬念拖延** | 同一悬念设了30章+不回收 | ★★★ |

### 检查方法

1. 扫描每章的情绪标签（正面/负面/中性）
2. 标记连续3章+低谷/中性区域
3. 标记信息密度过高的章节（超过3个新概念）
4. 标记无明确冲突的章节
5. 汇总为"高风险弃读点"列表

## 四、金手指节奏

### 检查标准

| 检查项 | 合格标准 |
|--------|---------|
| 首次展示 | 前 3 章内必须展示金手指 |
| 升级间隔 | 每 15-25 章有可见的能力提升 |
| 升级方式 | 不是单纯"练级"，需要与剧情/认知关联 |
| 瓶颈期 | 可以有但不超过 20 章 |
| 终极能力 | 不超过最终战前 10 章获得 |

### 能力曲线检查

```
实力
 ↑
 |                              ★终极
 |                    ★五阶
 |              ★四阶
 |        ★三阶
 |   ★二阶
 | ★一阶
 +──────────────────────────────→ 章节
   V1    V2    V3    V4    V5    V6
```

检查：是否有"长期平坦"区？是否有"突然跳级"？
