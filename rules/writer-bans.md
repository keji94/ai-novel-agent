# 写作禁令清单

> Writer 每章写作前必读。完整替换表见 `rules/replacements/ai-traces.yaml`。

## 绝对禁止（critical）

- 禁止"首先...其次...最后"列表式叙述 → 直接叙述，自然过渡
- 禁止"全场震惊/全场哗然"集体反应套话 → 具体描写每个人的不同反应

## 高频 AI 痕迹词（写作时主动替换）

| 禁止 | 替换为 |
|------|--------|
| 不得不承认 | 说实话 |
| 毋庸置疑 | 毫无疑问 |
| 显而易见 | 一眼可见 |
| 仿佛 | 像（少用如同） |
| 不禁 | 删除直接描写 / 没忍住 / 下意识 |
| 竟然 | 删除或换其他表达 |

## 写作习惯禁令

- 禁止解说式旁白（"不是X，而是Y"句式）→ 直接陈述结果
- 禁止同一信息重复展开 → 只在最合适的时机展开一次
- 禁止主角长期沉默 → 主角必须参与对话，哪怕简短回应
- 明喻密度控制：每千字 ≤ 2 个"像/像是"比喻，不是每个感受都需要配比喻 → 挑最有冲击力的 1-2 个用，其余直接叙述
- 禁止编号式自我总结：不要给心理经验排号（"第X件事/第X课/第一个感受"）→ 让顿悟自然融入叙事，不要给读者列清单
- 禁止反标签句式变体：不要"没有X，没有Y。不是不X，是Z"这种先否定再解释的句式 → 直接写角色的真实感受，不用先否定一堆不存在的状态
- 感官聚焦：每段最多 1-2 种感官，根据角色情绪选择主导感官 → 恐惧用听觉+触觉，平静用视觉，不要五官全开式描写
