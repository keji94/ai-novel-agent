## 总体原则

> 世界观不是静态说明书，而是动态运转系统——读者看到的不是设定，而是"这台手表碾压主角的命运"。

---

## 技巧一：推演术

- **核心原理**：从一个核心设定出发，向各维度穷尽推演，让世界细节自然生长。
- **具体操作步骤**：
  1. 确定一个反常/极端的核心设定（如"世界从未停过雨"）
  2. 从建筑、交通、食物、阶层、文化等维度逐一推演
  3. 让每个维度的推演结果互相影响，形成连锁反应
- **适用场景**：创建差异化世界观的初始阶段；需要一个"不一样"的基底设定时
- **示例**：核心设定"世界一直在下雨从未停过" → 高脚楼/漂浮建筑 → 水路交通+飞行兽 → 水上种植 → 船王为贵族 → 崇拜雨神、太阳成神话
- **注意事项/常见误区**：
  - 只推演一两个维度就停手，世界单薄
  - 推演时忽略维度间的因果联动，设定之间互相矛盾
  - 追求面面俱到，堆砌设定（"扔一颗石子，波纹自己会荡开"即可）

---

## 技巧二：稀缺术

- **核心原理**：稀缺是驱动人类行为的第一动力，确定世界中的稀缺物就能自动生成权力结构和冲突。
- **具体操作步骤**：
  1. 确定所有人都在争夺的稀缺物（资源型/资格型/知识型）
  2. 推导：稀缺物 → 垄断者 → 规则制定 → 底层挣扎反抗
  3. 将主角置于这条链条的矛盾位置
- **适用场景**：设计核心矛盾线；让世界观与主线剧情产生有机联系
- **示例**：
  - 资源型：灵果、天材地宝
  - 资格型：内城居住权、修炼资格
  - 知识型：破解永生的古老文字
- **注意事项/常见误区**：
  - 稀缺物太抽象（如"幸福"），缺乏具体争夺感
  - 稀缺物与剧情脱节，设定里存在但主角不涉及
  - 多个稀缺物互相冲突未处理优先级

## 技巧三：枷锁术

- **核心原理**：不是给角色自由，而是给角色枷锁——世界默认的价值观本身就是最强大的剧情驱动力。
- **具体操作步骤**：
  1. 定义这个世界的集体潜意识/默认价值观（崇尚什么、鄙视什么）
  2. 将主角置于与这套价值观的冲突位置
  3. 让"遵守"和"反抗"都有代价，形成两难
- **适用场景**：设计角色内心冲突；让配角的行为有集体逻辑而非"反派脸谱化"
- **示例**：
  - 崇尚血亲复仇的文明 → 主角必须复仇，不是选择而是枷锁
  - 笑贫不笑娼的商业文明 → 破产后所有人翻脸，主角只能靠自己
  - 武力至上的文明中的智者 → 价值观矛盾撑起一本书
- **注意事项/常见误区**：
  - 世界观价值观模糊，角色没有必须遵守或反抗的东西
  - 枷锁太弱，主角可以轻松无视，缺乏张力
  - 所有角色价值观完全一致，缺乏内部多样性

## 技巧四：三术联动法（综合应用）

- **核心原理**：推演术搭骨架、稀缺术埋冲突、枷锁术压角色，三术叠加让世界观和剧情自动生长。
- **具体操作步骤**：
  1. 用推演术确定一个核心设定（如"世界被迷雾笼罩，阳光稀缺"）
  2. 用稀缺术确定争夺焦点（如"所有人争夺能驱散迷雾的光石"）
  3. 用枷锁术确定默认价值观（如"见光者被视为神明使者，拥有最高地位"）
  4. 让主角同时触碰三者 → 剧情自然爆发
- **适用场景**：从零构建完整世界观；大纲阶段验证世界观能否撑起长篇
- **示例**：
  - 推演：世界被迷雾笼罩，阳光稀缺
  - 稀缺：所有人争夺能驱散迷雾的光石
  - 枷锁：见光者被视为神明使者
  - → 主角偷光石 → 发现遗迹 → 质疑见光者 → 被追杀，剧情自然生长
- **注意事项/常见误区**：
  - 三术割裂使用，推演的世界和稀缺物没有逻辑关联
  - 枷锁与稀缺物矛盾（如稀缺物不重要但所有人都想要）
  - 没有验证联动是否能产出冲突，世界观自洽但缺乏故事性
