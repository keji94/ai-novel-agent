# 推演术

## 技巧描述

从一个核心设定出发，向各维度穷尽推演，让世界细节自然生长。核心理念是"扔一颗石子，波纹自己会荡开"，而非堆砌设定。

## 核心要点

1. **确定核心设定**：选择一个反常/极端的核心设定（如"世界从未停过雨"）
2. **多维度推演**：从建筑、交通、食物、阶层、文化等维度逐一推演
3. **因果联动**：让每个维度的推演结果互相影响，形成连锁反应

## 应用场景

- 创建差异化世界观的初始阶段
- 需要一个"不一样"的基底设定时
- 大纲阶段搭建世界观骨架

## 示例

> 核心设定"世界一直在下雨从未停过" → 高脚楼/漂浮建筑 → 水路交通+飞行兽 → 水上种植 → 船王为贵族 → 崇拜雨神、太阳成神话

**分析**: 从一个自然现象出发，连锁推演出建筑形态、交通方式、经济结构、社会阶层、宗教信仰，形成自洽的世界观链条。

## 综合应用（三术联动法）

推演术搭骨架、稀缺术埋冲突、枷锁术压角色，三术叠加让世界观和剧情自动生长。

**操作步骤**：
1. 用推演术确定一个核心设定（如"世界被迷雾笼罩，阳光稀缺"）
2. 用稀缺术确定争夺焦点（如"所有人争夺能驱散迷雾的光石"）
3. 用枷锁术确定默认价值观（如"见光者被视为神明使者，拥有最高地位"）
4. 让主角同时触碰三者 → 剧情自然爆发

**联动示例**：
- 推演：世界被迷雾笼罩，阳光稀缺
- 稀缺：所有人争夺能驱散迷雾的光石
- 枷锁：见光者被视为神明使者
- → 主角偷光石 → 发现遗迹 → 质疑见光者 → 被追杀，剧情自然生长

## 注意事项

- 只推演一两个维度就停手，世界单薄
- 推演时忽略维度间的因果联动，设定之间互相矛盾
- 追求面面俱到，堆砌设定（"扔一颗石子，波纹自己会荡开"即可）
- 三术联动时需确保推演的世界与稀缺物、枷锁有逻辑关联
