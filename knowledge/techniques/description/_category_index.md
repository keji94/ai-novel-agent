# description 分类索引

> 描写技巧：环境、动作、心理、感官

## 技巧列表

### T010 禁用破折号
- 评分: 5 | 难度: 1
- 一句话: 章节正文中不得出现破折号（——），用逗号、句号、冒号、省略号等替代
- 场景: 所有章节正文写作
- 标签: 标点规范, 正文禁用

### T011 禁用解说式旁白
- 评分: 5 | 难度: 2
- 一句话: 不要用"不是X，是Y"格式做旁白解说，直接写结果
- 场景: 情绪描写、事实对比、心理活动
- 标签: 旁白技巧, 避免解说, 直接表达

### T012 动作描写指代清晰
- 评分: 4 | 难度: 2
- 一句话: 动作描写的指代对象超过一句就要重新交代，确保读者无需回溯
- 场景: 动作描写、身体部位指代、物品指代
- 标签: 指代清晰, 动作描写, 可读性

### T019 感官聚焦
- 评分: 5 | 难度: 3
- 一句话: 每段描写最多聚焦1-2种感官，不要一次性视觉听觉嗅觉全部拉满
- 场景: 怪物/环境/战斗等需要感官描写的段落
- 标签: 感官描写, 描写节制, 避免AI全开
