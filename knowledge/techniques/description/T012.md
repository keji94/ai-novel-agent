# T012 动作描写指代清晰

- 评分: 4/5 | 难度: 2/5
- 一句话: 动作描写的指代对象超过一句就要重新交代，确保读者无需回溯
- 场景: 动作描写、身体部位指代、物品指代
- 标签: 指代清晰, 动作描写, 可读性

## 详细说明
动作描写要确保读者不需要回溯就能理解完整画面。当一个指代对象（手、物品等）隔了一段文字才再次被使用时，读者可能已经忘了指什么，此时应重新交代完整描述而非用代词。

## 正面示例
"捂在嘴上的手"隔了一段后再次提及，仍写"捂在嘴上的手"而非"那只手"。

## 反面做法
第一句写"一只粗糙的手捂住了他的嘴"，隔了三段后写"那只手松开了"，读者需要回溯才能确认是哪只手。
