# T019 感官聚焦

- 评分: 5/5 | 难度: 3/5
- 一句话: 每段描写最多聚焦 1-2 种感官，不要一次性视觉听觉嗅觉全部拉满
- 场景: 怪物/环境/战斗等需要感官描写的段落
- 标签: 感官描写, 描写节制, 避免AI全开

## 详细说明
AI 写作的经典特征是"感官全开"——在同一段甚至同一句中，视觉、听觉、嗅觉、触觉、味觉全部给出。这看起来"描写丰富"，实际读感是：信息过载，没有焦点，不真实。

人类的感知是有选择的。人在恐惧时，听觉会异常敏锐但视觉可能模糊；人在疼痛时，触觉占主导但嗅觉退居其次。写作应模拟这种**选择性感知**，而非机械罗列五种感官。

原则：
1. 每段聚焦 1-2 种感官，最多不超过 3 种
2. 根据角色情绪状态选择主导感官（恐惧→听觉，愤怒→视觉/触觉）
3. 留白比写满更有力——没写到的感官让读者自己补

## 正面示例
```
苏辰没忍住，偏头看了一眼。三只两米多高的铁灰色节肢动物围着一具尸体，
口器像交叠的剃刀，一张一合间，肋骨断裂的声音清清楚楚地传过来。
```
→ 只聚焦视觉+听觉，嗅觉留给后面另起一段。

## 反面做法
```
三只两米多高的铁灰色节肢动物围着一具尸体，口器像交叠的剃刀，
一张一合间，肋骨断裂的声音清清楚楚地传过来。
暗红色的体液在碎石间蔓延，散发着甜腻的腥臭。
```
→ 视觉+听觉+嗅觉一句话内全给，AI式"感官全开"。
