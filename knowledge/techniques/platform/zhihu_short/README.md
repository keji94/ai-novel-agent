# 知乎短篇写作技巧

> 此目录存放从知乎短篇范文中学习到的写作技巧

## 目录结构

```
zhihu_short/
├── README.md           # 本文件
├── opening.md          # 开篇技巧
├── pacing.md           # 节奏把控
├── ending.md           # 结尾技巧
├── character.md        # 人物塑造
└── suspense.md         # 悬念设置
```

## 技巧来源

技巧通过以下方式学习：
1. 用户分享知乎短篇范文链接
2. Analyst 解析内容并提取技巧
3. Learner 入库并建立索引

## 使用方式

写作知乎短篇时：
1. Writer 自动加载 `category_mapping.json` 中匹配的技巧
2. 或手动调用 `search_learned_techniques("知乎短篇")`

## 技巧编号规则

- 格式：`T-ZHIHU-{序号}`
- 示例：`T-ZHIHU-001`、`T-ZHIHU-002`
