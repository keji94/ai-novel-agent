# 情感弧线

追踪主要角色的情绪变化和成长轨迹。

## 弧线类型

- **成长弧**: 角色从弱到强，从幼稚到成熟
- **堕落弧**: 角色从善到恶，从高到低
- **觉醒弧**: 角色从迷茫到明确目标
- **复仇弧**: 角色从受害到反击
- **救赎弧**: 角色从罪人到赎罪

## 记录格式

```yaml
角色: 林风
弧线类型: 成长弧
起点状态: 练气期，被欺压，渴望变强
当前状态: 筑基初期，初露锋芒，自信心增强
目标状态: 化神期，宗门核心，成为一方强者
关键事件:
  - 第1章: 觉醒签到系统，获得希望
  - 第10章: 突破筑基期，实力大增
情绪变化:
  - 第1章: 压抑 → 惊喜 → 期待
  - 第10章: 紧张 → 突破的喜悦 → 自信
```

## 注意事项

1. **情绪连续性**: 每章的情绪状态要承接上一章
2. **成长节奏**: 成长要有铺垫，不能突兀
3. **弧线终点**: 每个角色要有明确的成长目标
4. **多弧线交织**: 主角和配角的弧线要相互影响
