# 资源账本

追踪所有数值型资源：物品、金钱、修炼资源等。

## 记账规则

1. **期初余额**: 从上一章结转
2. **增量**: 本章获得/消耗的资源，逐笔列出
3. **期末余额**: 期初 + 增量 = 期末
4. **衰减规则**: 同质资源重复炼化必须写明衰减

## 资源分类

### 修炼资源
- 灵石: 数量、品级
- 丹药: 名称、数量、功效
- 功法: 名称、品级、来源

### 法宝装备
- 法宝: 名称、品级、状态
- 武器: 名称、品级、状态
- 防具: 名称、品级、状态

### 通用资源
- 金钱: 数量、单位
- 物品: 名称、数量、用途
