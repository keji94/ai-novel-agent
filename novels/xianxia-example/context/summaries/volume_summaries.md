# 卷摘要集

## 第一卷：青云崛起（第1-70章）

### 卷简介

林风穿越后获得签到系统，从外门弟子起步，经历宗门内斗，最终成为内门弟子，开启修仙之路。

### 主线剧情

1. **开篇阶段**（1-10章）：穿越觉醒，签到系统激活
2. **成长阶段**（11-30章）：每日签到，实力稳步提升
3. **冲突阶段**（31-50章）：与李明势力冲突，陷入危机
4. **高潮阶段**（51-70章）：外门大比，一鸣惊人，进入内门

| 角色 | 初始状态 | 卷末状态 |

| 李明 | 练气五层 | 被逐出宗门 |
