# 示例项目：仙道长生

## 项目信息

- 类型：仙侠
- 状态：示例
- 创建时间：2024-01-15

## 目录说明

本目录为示例项目结构，展示小说项目应包含的文件：

```
xianxia-example/
├── project.json          # 项目元数据
├── settings/             # 世界观设定
│   ├── world.md          # 世界观
│   ├── power_system.md   # 修炼体系
│   └── factions.md       # 势力设定
├── characters/           # 角色设定
│   └── main.md           # 主角设定
├── outline/              # 大纲
│   └── main.md           # 总大纲
└── chapters/             # 章节
    └── chapter_001.md    # 第一章
```

## 使用方式

1. 复制此目录作为新项目模板
2. 修改 project.json 中的项目信息
3. 在各目录中填写对应内容

## 注意事项

- 设定文件使用 Markdown 格式
- 章节文件命名格式：chapter_{n}.md
- 保持设定的一致性，避免前后矛盾
