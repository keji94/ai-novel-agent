## 四、异族设定

### 异族来源与终极目标

异族来自裂隙彼岸的异世界——一个以"吞噬文明"为本能的异界种族。

**异界的本质——"文明黑洞"**：
- 异界不像地球文明那样创造和积累，而是以吞噬其他世界的文明记忆来维持自身存在。异界是一个"文明黑洞"，它需要不断进食文明才能存续
- 界锚是异界感知到地球上密集的文明记忆后投放的入侵工具——一枚钉入时空壁垒的钉子，用来打开通道
- 异界本身可能存在多个"文明黑洞"，地球只是其中之一的猎物

**异族的终极目标——"文明牧场"**：
- 血裔族高层（后世称为"王座"）的长期战略并非"消灭人类"，而是**将地球变成一座"文明牧场"**
- 完全消灭人类 = 断绝文明记忆的来源 = 断粮。异族需要人类活着、创造文明、积累文明记忆，然后由异族定期"收割"
- 最理想的状态是：人类被圈禁在有限的聚集地中，维持最低限度的文明活动（制造工具、建立秩序、传承文化），文明记忆在低水平上持续积累。当积累到一定密度时，异族发动收割
- 这解释了异族120年来所有看似矛盾的行为模式

### 异族120年战略演变

异族的每一步行动都有明确的战略逻辑，从未"降智"或"莫名其妙停止"：

| 时期 | 异族行为 | 战略目的 |
|------|----------|----------|
| 纪元1-5年 | 全面入侵、毁灭城市 | **破毁期**：摧毁人类现代文明体系，消灭高密度文明记忆节点（城市），迫使文明记忆向古建筑遗址集中 |
| 纪元5-10年 | 追杀迁徙中的难民 | **驱赶期**：将幸存者驱赶至古建筑避难所，让人类文明记忆集中在可控的"点"上 |
| 纪元10-20年 | 持续围攻避难所 | **试探期**：测试古物共鸣的威力，评估人类威胁等级。发现古物共鸣有效克制后，开始调整战略 |
| 纪元20-35年 | 发动深渊巨兽、猩红之冬 | **消耗期**：消耗人类高阶共鸣者和古物储备。深渊巨兽是"重锤"，用来砸碎最强的聚集地 |
| 纪元35-55年 | 收缩攻势，转向渗透 | **评估期**：正面强攻代价太大，转用塑形族渗透制造内乱，同时评估人类各聚集地的实力分布 |
| 纪元55-65年 | 进攻骤减，营造"和平" | **陷阱期（围困战略启动）**：停止大规模进攻，让人类放松警惕、人口膨胀、资源消耗加速。同时暗中在交通要道部署封锁线 |
| 纪元65-75年 | 切断聚集地间联系、驱赶小聚集地人口 | **切割期**：封锁线成型，将人类切割成孤岛。有选择地攻击小聚集地，将人口向大聚集地驱赶集中 |
| 纪元75-90年 | 建立常态化封锁线 | **围困期**：封锁线完全成型，各聚集地彻底孤立。异族转入"围而不攻"模式，只需守住封锁线 |
| 纪元90-120年 | 消耗战（猎杀共鸣者、骚扰拾荒队） | **牧场期**：维持围困，通过持续消耗战（猎杀共鸣者、古物死寂）让人类在有限资源中逐渐衰退。等待文明记忆积累到可收割密度 |

### 主要异族

**1. 虫潮族（最常见）**
- 外形：巨型昆虫，从蚂蚁到甲虫各变种
- 特点：数量庞大、繁殖极快、低级个体无智力但受高阶异族指挥
- 等级：工虫→兵虫→将虫→虫母
- 弱点：怕火、怕震
- **战略角色**：封锁线的巡逻兵、消耗战的炮灰、驱赶人类的牧羊犬

**2. 血裔族（高阶敌人）**
- 外形：人形但非人，皮肤苍白，瞳孔为红
- 特点：高智商、可操控低等异族、拥有"血术"
- 等级对应人类共鸣等级
- 弱点：阳光（但可适应）、古物共鸣之力
- **战略角色**：封锁线指挥官、战略规划者、对人类社会的渗透者。血裔族的"王座"是整个入侵战略的制定者
- **"王座"的存在**：异族最高指挥层，由数名最高阶血裔族组成。它们不直接参与战斗，而是制定长期战略。王座成员可能从未被人类见到过

**3. 塑形族（特殊敌人）**
- 外形：可变形，能模仿人类
- 特点：渗透、间谍、制造内乱
- 极度危险：无法被常规手段识别
- 弱点：古物共鸣者可感知其"非人气息"
- **战略角色**：信息战的主力。渗透人类聚集地制造内乱、窃取情报、暗杀关键人物。围困战略中，塑形族负责确保人类各聚集地之间无法建立信任

**4. 深渊巨兽（BOSS级）**
- 外形：各种巨型生物
- 特点：移动要塞级别，一次出现可毁灭聚集地
- 人类最强战力才能应对
- 弱点：核心（体内关键部位）
- **战略角色**：战略威慑武器。当某个聚集地的古物共鸣力场过强、威胁到封锁线时，深渊巨兽被用来"重置"该聚集地的防御

### 异族与古物的关系
- 异族本能地厌恶古物，因为古物承载的"文明记忆"对它们有天然克制
- 这也是人类能存活的关键原因
- 但高级异族已学会对抗古物之力，且在持续进化中

### 异族封锁线体系

异族围困战略的核心执行机制：

**封锁线构成**：
- **虫潮巡逻带**：在聚集地外围50-200公里范围内，虫潮以10-30只为一组持续巡逻，覆盖所有已知道路和通道
- **血裔族指挥哨**：每条封锁线由1-3名血裔族军官指挥，负责战略调度和关键时刻的出击
- **关键节点要塞**：在山脉隘口、桥梁、峡谷等必经之路上，异族建立了永固据点（利用异界能量改造地形）
- **信息封锁**：塑形族定期向各聚集地投放虚假情报，夸大其他聚集地的危险程度，阻止人类尝试建立联系

**封锁效果**：
- 聚集地之间无法通行：商队出发后几乎必然遭遇虫潮伏击
- 拾荒范围受限：三环外200公里以外的区域对人类来说已是"禁区"
- 信息完全隔绝：各聚集地不知道其他聚集地是否还存在
- 人员流动为零：没有任何合法途径离开或进入聚集地
