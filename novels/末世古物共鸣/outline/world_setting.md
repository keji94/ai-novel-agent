# 世界观设定：末世古物共鸣

## 基本信息
- 作品类型：末世废土 + 异能进化
- 核心卖点：古物共鸣觉醒体系 / 底层逆袭 / 博物馆保安的文物知识碾压
- 整体风格：硬核求生、智斗布局、热血燃点
- 末世时间线：异变至今约120年（主角穿越时）

---

## 一、世界背景

### 异变纪元（纪元前0年）
- 一颗来历不明的陨石坠入太平洋，未造成直接破坏
- 但陨石释放的未知能量波及全球，被称为"源潮"
- 源潮持续三年，地球生物基因被深度改写

### 灾厄降临（纪元1-10年）
- 纪元1年：异族从"裂隙"中入侵，裂隙出现在古文明遗址附近
- 纪元2-3年：异族如潮水般涌出，人类正规军全线溃败。全球人口从80亿骤降至不足15亿。核武器对中低级异族有杀伤力，但引发连锁反应——核爆的辐射被源潮催化，反而加速了部分异族的变异。各国政府在纪元3年内陆续瓦解
- 纪元4-5年：末世前最后一批有组织的抵抗——"方舟计划"，试图用古建筑力场建立避难所。计划因内部叛变（后被证实为塑形族渗透）而半数失败，幸存的避难所成为日后聚集地的雏形
- 纪元6-8年：血裔族正式登场，展现出远超虫潮族的智慧和组织力。它们开始操控虫潮，将零散的异族入侵升级为有战略目标的"灭族战争"。人类被迫从城市撤退至古建筑避难所，史称"大迁徙"
- 纪元9-10年：全球人口跌至不足5亿。所有现代国家机器彻底消失，工业体系全面瘫痪。人类进入最黑暗的时代

### 黑暗纪元（纪元10-40年）
- **人类最低谷**，被称为"长夜三十年"
- 纪元10-15年：古建筑避难所勉强维持，食物短缺、疾病肆虐、异族围困。多个避难所因力场耗尽而沦陷，数以百万计的人被屠杀或成为异族的食物。幸存者开始真正理解古物共鸣——第一批触物者在绝望中觉醒
- 纪元15-20年：共鸣体系的雏形被摸索出来。有人发现与古物共鸣能获得对抗异族的力量。这个发现如同一道微光，点燃了人类的希望。但早期共鸣者缺乏系统理论，觉醒率极低，且大量人因不当共鸣而精神崩溃
- 纪元20-25年：**第一次人类反击**。长安城前身——以古城墙为核心的避难所，在一位传奇共鸣者（后世称为"城墙老祖"）的率领下，利用城墙建筑共鸣力场击退了一次虫潮围城。这是人类第一次在正面战场上击退异族的大规模进攻，极大鼓舞了士气。此后各避难所纷纷效仿，建立以古建筑为核心的防御体系
- 纪元25-30年：深渊巨兽首次出现。一头体型堪比山脉的巨兽摧毁了当时第三大的聚集地"金陵堡"，三十万人在一日之间化为灰烬。人类意识到异族的威胁远超想象。但这次惨痛的失败也催生了一个关键发现——深渊巨兽的核心可以作为超级古物，赋予共鸣者远超常人的力量
- 纪元30-35年：**血裔族大举入侵**。血裔族发动了被称为"猩红之冬"的总攻，数十个聚集地同时遭到血裔族精锐部队的突击。人类损失惨重，但也在战斗中积累了宝贵的经验。几位高阶共鸣者联手击杀了一名血裔族领主级存在，这是人类第一次击杀高阶血裔族
- 纪元35-40年：长夜将尽。人类开始系统化地研究共鸣体系，初步建立了等级制度和流派划分。聚集地之间开始建立联系和贸易路线。匠造派的前身——一群擅长利用古物碎片制作工具的工匠——开始形成组织

### 复兴纪元（纪元40-80年）
- **文明重建的四十年**
- 纪元40-50年：聚集地大规模建立，人类进入"圈地扩张"期。共鸣体系趋于成熟，共鸣者数量快速增长。三环结构成为聚集地的标准模式。各大势力（共鸣者公会、匠造协会、商会）的雏形在这一时期形成
- 纪元50-55年：**塑形族渗透危机**。塑形族利用其伪装能力，成功渗透进多个聚集地的核心管理层，引发了一连串内部清洗和信任危机。史称"影祸之乱"。此后各聚集地建立了严格的身份验证制度，对陌生人高度警惕——这也导致聚集地之间的人员流动大幅减少，势力格局开始固化
- 纪元55-60年：人类在第一次深渊巨兽危机后，组织了一次有史以来最大规模的联合猎杀行动，成功击杀了一头深渊巨兽。巨兽核心被用于强化长安城的城墙力场，长安城由此跃升为最大聚集地
- 纪元60-65年：**大和平期**。异族入侵出现了一个罕见的低潮期，裂隙活动显著减少（原因至今不明）。人类利用这段喘息时间大力发展，聚集地人口翻倍。但和平也让内部矛盾激化——资源分配不公、等级制度森严、三环外的底层人民生活困苦
- 纪元65-70年：信仰教派在这一时期蓬勃发展。"古物崇拜"成为最主流的信仰，将古物视为"先祖之灵"的载体。一些极端教派甚至认为古物共鸣是"神授之力"，共鸣者是"先知"。温和派则将博物馆视为圣地，认为那里保存着"失落文明"的终极秘密
- 纪元70-75年：**第二次异族大入侵**。裂隙重新活跃，且规模远超前次。这次异族明显更加"聪明"——它们学会了绕过聚集地的防御弱点，甚至学会了围点打援。人类被迫放弃了约三分之一的聚集地。但经过几十年发展的人类已非吴下阿蒙，战线最终稳住
- 纪元75-80年：战后重建。人类深刻反思防御体系，加强了聚集地之间的协作。匠造协会在这一时期研发出多种古物衍生武器，大幅提升了人类的整体战力。四大共鸣流派正式确立并被体系化传承

### 稳固期（纪元80-120年）
- **格局固化的四十年**
- 纪元80-90年：聚集地之间的势力格局基本定型。长安城成为公认的最大聚集地，但内部权力斗争激烈。三环内外的鸿沟已固化为不可逾越的社会阶层。共鸣者世家的概念形成——高阶共鸣者的后代觉醒率更高，"血统论"开始流行
- 纪元90-95年：小规模异族入侵成为常态，人类已能从容应对。但裂隙中出现了一些前所未有的"变异异族"，它们比普通异族更强，似乎在加速进化。有学者（文心派）推测异族也在"学习"
- 纪元95-105年：知识断代问题日益严重。末世前亲历者已全部离世，"黄金纪元"（即我们现在的时代）变成了半真半假的传说。人们对末世前的科技认知极度扭曲——有人把手机当成"神谕之镜"，有人把飞机残骸当成"先祖的钢铁巨鸟"，博物馆里的展品则被视为"失落文明的圣物"。真正的科技知识只剩下碎片化的口耳相传，且大部分已被严重误传
- 纪元105-115年：长安城内部矛盾激化。三环外人口膨胀，资源紧张，多次爆发小规模暴动。共鸣者公会加强了镇压，同时通过"共鸣者征召制"从三环外选拔有潜力的年轻人，既补充战力又缓解矛盾——但本质上只是让底层人民多了一条"向上"的窄路
- 纪元115-120年：**当前局势**。人类在各大聚集地中维持着脆弱的平衡。异族入侵持续但频率降低，更多是低级虫潮骚扰和偶发的血裔族袭击。表面和平下暗流涌动——裂隙深处的异族似乎在酝酿什么，塑形族的渗透从未停止，聚集地内部的权力斗争愈演愈烈

### 当前时代（纪元120年）
- 人类已建立数百个聚集地，最大者可容纳百万人
- 但异族从未停止入侵，人类仍处于生死存亡的边缘
- 末世前亲历者已全部离世，知识断层极度严重
- 聚集地社会阶层固化，三环内外如两个世界
- 主角穿越的节点

## 二、古物共鸣体系

### 核心规则
- **共鸣条件**：人需与古物产生精神共振，需对古物有"理解"和"感悟"
- **古物定义**：年满50年以上且承载人类文明记忆的器物、建筑、遗迹
- **共鸣媒介**：古物的"文明记忆"——它所承载的历史、故事、情感
- **共鸣代价**：过强共鸣会导致精神侵蚀（古物中的执念、怨念反噬）

### 共鸣等级（共鸣者）

| 等级 | 称号 | 共鸣深度 | 能力特征 | 人数占比 |
|------|------|----------|----------|----------|
| 一阶 | 触物者 | 表面共鸣 | 获得古物一项特性 | 70% |
| 二阶 | 融物者 | 浅层共鸣 | 古物特性强化+外放 | 20% |
| 三阶 | 化物者 | 深度共鸣 | 可化用古物之力为招式 | 7% |
| 四阶 | 御物者 | 灵魂共鸣 | 可同时共鸣多件古物 | 2% |
| 五阶 | 通灵者 | 本源共鸣 | 与古物"对话"，解锁隐藏信息 | 0.8% |
| 六阶 | 造化者 | 创造共鸣 | 可赋予物品"古物属性" | 0.1% |
| 七阶 | 万物共鸣 | — | **传说境界**，人类历史上从未有人真正达到。五阶通灵者已是人类已知极限，六阶造化者仅存理论记载。七阶被认为需要与大地本身共鸣，可能需要血裔族的血脉辅助——这也是异族和混血受到关注的原因之一 | 仅存传说 |

### 共鸣类型

1. **武兵共鸣**：刀剑、铠甲、兵器 → 战斗力提升
2. **器物共鸣**：器皿、工具、乐器 → 辅助/特殊能力
3. **建筑共鸣**：古建筑、城池、桥梁 → 防御/领域能力
4. **书卷共鸣**：古籍、画卷、碑帖 → 知识/精神力
5. **遗物共鸣**：名人遗物、圣物 → 特殊/稀有能力

### 共鸣流派
- **武道派**：以兵器共鸣为主，强调战斗
- **匠造派**：以器物共鸣为主，擅长炼制
- **御城派**：以建筑共鸣为主，守护聚集地
- **文心派**：以书卷共鸣为主，精神系能力

## 三、异族设定

### 异族来源
- 来自裂隙彼岸的异世界种族
- 源潮是它们入侵的前奏，裂隙是它们的通道
- 它们以"吞噬文明"为食——越文明的种族，越美味

### 主要异族

**1. 虫潮族（最常见）**
- 外形：巨型昆虫，从蚂蚁到甲虫各变种
- 特点：数量庞大、繁殖极快、低级个体无智力
- 等级：工虫→兵虫→将虫→虫母
- 弱点：怕火、怕震

**2. 血裔族（高阶敌人）**
- 外形：人形但非人，皮肤苍白，瞳孔为红
- 特点：高智商、可操控低等异族、拥有"血术"
- 等级对应人类共鸣等级
- 弱点：阳光（但可适应）、古物共鸣之力

**3. 塑形族（特殊敌人）**
- 外形：可变形，能模仿人类
- 特点：渗透、间谍、制造内乱
- 极度危险：无法被常规手段识别
- 弱点：古物共鸣者可感知其"非人气息"

**4. 深渊巨兽（BOSS级）**
- 外形：各种巨型生物
- 特点：移动要塞级别，一次出现可毁灭聚集地
- 人类最强战力才能应对
- 弱点：核心（体内关键部位）

### 异族与古物的关系
- 异族本能地厌恶古物，因为古物承载的"文明记忆"对它们有天然克制
- 这也是人类能存活的关键原因
- 但高级异族已学会对抗古物之力

### 异族攻守平衡（120年演变）
- **纪元1-20年**：人类全面溃退，被动挨打。异族占据绝对优势
- **纪元20-30年**：人类开始反击，但只是防守反击。深渊巨兽的出现打破了脆弱的平衡
- **纪元30-40年**：血裔族大入侵将人类再次推到悬崖边，但人类在绝境中学会了协同作战
- **纪元40-60年**：人类进入战略相持阶段。能守住既有聚集地，但无力大规模反攻
- **纪元60-80年**：大和平期让人类得以发展，但第二次大入侵证明异族只是在积蓄力量
- **纪元80-120年**：僵持阶段。人类和异族形成了一种诡异的平衡——异族不再发动灭国级入侵，而是以骚扰和渗透为主；人类也不再试图关闭裂隙，而是维持现状。双方似乎都在等待什么

## 四、社会结构

### 聚集地结构

```
          ┌─────────┐
          │ 核心区域  │ ← 古建筑本体所在，权贵居住区
          │ （内城）  │    最强防御，资源最丰富
          └────┬────┘
               │
       ┌───────┴───────┐
       │   二环区域     │ ← 中产/共鸣者居住区
       │   （商住区）    │    有基础防御设施
       └───────┬───────┘
               │
       ┌───────┴───────┐
       │   三环区域     │ ← 底层/无共鸣者居住区
       │   （贫民区）    │    防御薄弱，随时可能被放弃
       └───────┴───────┘
               │
        ┌──────┴──────┐
        │  三环外废墟  │ ← 拾荒者、流浪者
        │  （荒野区）   │    异族出没，朝不保夕
        └─────────────┘
```

### 120年演变下的社会特征
- **阶层固化**：经过几代人的发展，共鸣者世家已经形成。高阶共鸣者的后代觉醒概率更高，"血统论"深入人心。三环内的家族已把持权力数十年，底层几乎无上升通道
- **三环鸿沟**：三环内外不只是贫富差距，而是两个完全不同的世界。三环内的人可能一辈子没见过异族长什么样；三环外的人从记事起就在为生存挣扎。两边的价值观、语言习惯甚至信仰都已有明显差异
- **知识垄断**：共鸣者公会有意无意地垄断着关于古物共鸣的系统知识。文心派的书卷共鸣者掌握了大量末世前的残存文献，但他们只传授给"有资格"的人。底层人对古物的认知还停留在"神器崇拜"的阶段
- **代际遗忘**：120年意味着四到五代人。末世前亲历者已全部离世，他们的事迹在口耳相传中变得面目全非。当前时代的人对"黄金纪元"（我们的时代）的认知充满了神话化和误解

### 聚集地势力

**1. 共鸣者公会**
- 聚集地的实际掌控者
- 由高阶共鸣者组成，历经数代已形成世家格局
- 负责对外征伐、内部秩序
- 等级森严，低阶服从高阶
- 内部派系林立，暗斗不断

**2. 匠造协会**
- 炼制古物衍生品的专业组织
- 地位超然，各势力都需要的合作对象
- 内部有严格的师徒传承制度
- 掌握着大部分"实用技术"的知识，是末世中技术传承的核心力量

**3. 商会**
- 掌控聚集地的经济命脉
- 收购古物、出售物资
- 有自己的武装力量
- 部分商会家族的历史可追溯到复兴纪元初期

**4. 拾荒者**
- 三环外生存的人群
- 靠在废墟中搜寻古物和物资为生
- 主角的开局身份
- 社会地位最低，但最了解废墟
- 形成了自己的"废墟文化"和生存法则，三环内的人根本无法理解

**5. 信仰教派**
- 崇拜古物/崇拜祖先的宗教组织
- 各聚集地都有不同教派
- 有的温和、有的极端
- 120年间教派经历了多次分裂和融合，形成了复杂的信仰图谱
- 主流教派将博物馆视为"圣地"，认为其中保存着"先祖文明"的终极秘密

### 货币体系
- **源晶**：异族体内凝结的能量结晶，用途广泛
- **古物碎片**：古物碎裂后的残余，可用于低级共鸣催化
- 以物易物仍然普遍

### 核心聚集地（故事主要舞台）
- **长安城**（原型西安）：以古城墙为核心，最大的聚集地之一。城墙本身在源潮催化下产生了防御力场（建筑共鸣的自然结果），内城有简易电力供应和水循环系统，三环和三环外则无电力保障
- 故事从长安城外围的废墟开始

## 五、独特设定元素

### 源潮现象
- 仍在持续，时强时弱
- 强源潮期间，古物共鸣增强，异族也更强
- 被称为"潮汐日"

### 古物退化
- 部分古物因过度使用而"死寂"，失去共鸣能力
- 也有古物因特殊事件而"觉醒"，产生新的共鸣属性
- 造化者的能力正是赋予物品古物属性
- 120年来，大量古物已经死寂，可用的古物越来越少，这加剧了各势力对古物的争夺

### 信息断层（120年版）
- 末世前的历史大量失传，亲历者已全部离世
- "古代"（即我们现在的时代）被视为失落的黄金文明，但认知严重扭曲：
  - 手机 → "神谕之镜"，被认为能沟通先祖之灵
  - 飞机 → "先祖的钢铁巨鸟"，被当作某种神兽
  - 电脑 → "智慧之匣"，没人知道怎么用但都认为蕴含大智慧
  - 图书馆/博物馆 → "圣地"，保存着"失落文明的圣物"
  - 汽车 → "铁马车"，有的被拆解当材料，有的被完整保存当"圣物"
  - 互联网 → "先祖的意识之海"，被各种传说神话化
- 博物馆被视为圣地，但大多已被摧毁或洗劫一空。幸存的博物馆成为各势力争夺的战略资源
- 主角的现代知识成为降维打击——但他也发现，120年后的世界对"古代"的理解已经面目全非，他习以为常的知识在这里既是利器也是异类标志
- **智斗维度**：信息断代不仅是背景设定，更是剧情工具。主角可以利用信息差进行布局（例如他知道某处废墟原是博物馆而别人不知道；他知道某种化学反应可以制造简易炸药而匠造师们还在摸索）。反过来，末世原住民虽然科技知识缺失，但积累了120年的异族战斗经验和废墟生存智慧，在某些方面反而比主角更有经验。这种双向信息差为智斗提供了天然土壤——主角"降维打击"的同时也会因不了解末世规则而吃暗亏。120年的时间让这种信息差更加极端，也更加有趣

### 动物觉醒
- 受源潮影响，部分动物进化出智慧
- 有善有恶，可成为伙伴也可成为敌人
- 觉醒动物同样可与古物共鸣
- 120年间，觉醒动物已形成自己的生态圈和社会结构。部分高智商觉醒动物甚至与聚集地建立了"共生关系"
- **融入主线**：苏辰在三环外拾荒时收留一只觉醒的乌鸦（"墨羽"），它能传递简单信息和进行侦察，成为主角团队的重要辅助。墨羽在塑形族渗透事件中发挥了关键作用——动物对塑形族有天然的警觉反应，帮助苏辰提前识破了韩夜的伪装。铁牛后期也收留了一头觉醒的野猪作为坐骑。觉醒动物的存在让废墟生态更丰富，也为战斗和侦查增添了变数

## 六、角色调整说明

### 老孙头（需重新设计背景）
- **原设定**：末世前亲历者，亲历过异变
- **新方向建议**：老孙头可以是复兴纪元（纪元40-80年）出生的人，约60-70岁。他虽非末世亲历者，但年轻时曾跟随一位末世前亲历者的弟子学习，间接继承了一些"真正的"古物知识。他见过了太多虚假的"古物传说"，对那些将现代物品神化的教派嗤之以鼻。他对苏辰的态度之所以特殊，正是因为苏辰表现出了对古物的"正确理解"——这是他几十年只在师父身上见过的特质

### 苏辰身世谜团时间线
- 苏辰在纪元120年穿越，身体为末世原住民
- 推测苏辰出生于纪元100-105年左右（穿越时约15-20岁）
- 其母遇害应在纪元105-110年之间
- "裂隙之子"的设定意味着苏辰的出生与裂隙/异族有关，这个谜团需要在纪元100年左右的裂隙活动中找到对应
- 母亲的身份、遇害真相、与裂隙的关系——这些都需要在120年的时间线中找到合理的位置
