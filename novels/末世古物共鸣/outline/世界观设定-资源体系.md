## 六、资源衰减体系

### 核心矛盾：有限资源 vs 无限消耗

在异族封锁的孤岛格局下，每个聚集地都是一座资源孤岛。资源不是"分配不公"的问题，而是"总量不足"的问题。资源衰减是驱动一切内部矛盾和外部冲突的第一动力。

### 六大资源类别

#### 1. 原始矿物资源

| 资源 | 获取方式 | 现状（纪元120年） | 消耗用途 |
|------|----------|-------------------|----------|
| 铁矿 | 三环外废墟回收（拆解建筑、机械） | 越来越难找。近处废墟已被反复拾荒，远处的矿脉在封锁线外 | 武器、工具、建筑材料 |
| 铜矿 | 废墟回收（电线、管道、器物） | 近乎枯竭。铜是电传导关键材料，已被回收殆尽 | 古物衍生品的传导介质、匠造材料 |
| 硝石/硫磺/木炭 | 硝石（旧厕所/粪坑土法提取）、硫磺（温泉/矿物采集）、木炭（烧制） | 天然硝石矿和硫磺矿大多在封锁线外。土法硝石产量极低，品质不稳定。木炭因林木砍伐殆尽而趋紧 | 黑火药（军事、城墙防御、拾荒爆破开路）。匠造协会掌握最佳配比 |
| 盐 | 封锁线内盐湖/岩盐矿（极少）、废墟回收（末世前盐袋残余） | **极度稀缺**。无盐则食物无法保存、人体健康迅速恶化。封锁线内的盐源仅够内城配给，三环外居民长期缺盐 | 食物防腐（无冰箱时代唯一手段）、人体必需、皮革鞣制 |
| 药材 | 三环外采集（草药）、废墟回收（末世前药品残余） | 基础草药（止血草、退烧草）尚可维持，但成品药（抗生素、麻醉剂、缝合线）几乎耗尽。末世前药品残余已成为文心派和匠造协会的垄断物资 | 医疗救治、外伤处理、疫病防控。缺药是拾荒队死亡率居高不下的直接原因 |
| 石材/木材 | 三环外采集 | 尚可维持，但采集队频繁遭遇异族 | 建筑、燃料 |
| 煤炭 | 远处矿脉（大多在封锁线外） | 极度稀缺，只有长安城等少数聚集地能在封锁线内找到煤矿 | 燃料、冶炼 |

**三种必争资源**：盐、硝石/硫磺、药材是末世中最具争夺价值的矿物资源——技术门槛低（土法可制）、需求刚性（无替代品）、产地有限（多在封锁线外）。控制这三类资源的势力等于控制了生存权，因此也成为公会管控、私盐/私药黑市、拾荒者灰色交易的核心驱动。

**衰减趋势**：近处废墟已被反复拾荒数十年，可用物资越来越少。三环外的拾荒者需要走越来越远、冒越来越大的风险才能找到有价值的物资。预计50年内，大部分聚集地周边的废墟资源将被彻底耗尽。

#### 2. 古物资源

**古物的不可再生性**：
- 古物承载的"文明记忆"是有限资源。每次共鸣都会消耗古物中的文明记忆
- 当文明记忆消耗殆尽，古物进入"死寂"状态——失去一切共鸣能力，变成普通的旧物
- **古物死寂不可逆转**：没有任何已知方法能恢复死寂古物的共鸣能力
- 造化者（六阶）能"赋予"物品古物属性，但这是用自身的文明记忆去"填充"，代价极大且产出的古物品质远不如天然古物

**古物衰减数据（估算）**：

| 时期 | 可用古物数量（估算） | 备注 |
|------|---------------------|------|
| 纪元1年 | 极其丰富（全球数十亿件） | 人类文明数千年的积累 |
| 纪元40年 | 大量减少（战争消耗+死寂） | 黑暗纪元中大量古物被不当使用而死寂 |
| 纪元80年 | 严重不足 | 围困期开始后，无法从其他聚集地获取古物补充 |
| 纪元120年（当前） | **极度稀缺** | 长安城内可用的中高等级古物不足千件 |

**古物衰减对共鸣体系的影响**：
- 新共鸣者的觉醒越来越难——没有足够的古物供他们触发共鸣
- 现有共鸣者的进阶越来越慢——深度共鸣需要大量古物记忆的"喂养"
- 高阶共鸣者的培养周期从纪元40年的"10-15年"延长到纪元120年的"30年以上"
- **阶层固化加剧**：世家垄断了仅存的优质古物，底层人几乎不可能获得觉醒机会

#### 3. 能源

**电力现状**：
| 聚集地 | 电力状态 | 来源 | 覆盖范围 |
|--------|----------|------|----------|
| 长安城 | 有（极有限） | 水力发电（古物改良的水车装置） | 仅内城权贵区 |
| 铁壁城 | 无 | 无 | — |
| 圣光城 | 无（但有"古物照明"） | 火属性古物驱动的光源 | 教团核心区域 |
| 浮城 | 未知 | 可能利用潮汐或风力 | 未知 |

**非电力能源**：
- 薪柴：主要燃料，但长安城周边的林木已被砍伐殆尽，需要从越来越远的地方采集
- 动物油脂：低效但普遍
- 古物能源：火属性古物碎片可以提供热量，但消耗古物资源，不可持续
- **能源危机**：长安城内城每年冬天需要消耗大量燃料，三环外的居民则靠"挤在一起取暖"度过寒冬

#### 4. 食物

**食物来源**：
| 来源 | 占比（长安城估算） | 风险 |
|------|---------------------|------|
| 城内耕地 | 约40% | 产量低（土壤贫瘠、没有化肥）、面积有限 |
| 三环外采集/狩猎 | 约25% | 高风险（异族出没）、产量不稳定 |
| 水产（河流） | 约15% | 河流取水队需要武装护送，且有异族水生变种 |
| 三环外拾荒（末世前罐头/干货） | 约15% | 越来越少、越来越远 |
| 贸易/交换 | 约5% | 聚集地内部三环之间的物资流动 |

**衰减趋势**：
- 城内耕地因连续耕种数十年而土壤贫瘠，产量逐年下降
- 三环外的采集范围因异族封锁而缩小
- 末世前罐头和干货几乎已被搜刮殆尽
- **粮食缺口逐年扩大**，三环外居民长期处于半饥饿状态

#### 5. 水源

| 来源 | 状况 |
|------|------|
| 城墙内古井 | 建筑共鸣维持水质，水量有限，仅供内城和部分二环 |
| 城外河流 | 需要武装护送取水队，往返一次需2-3天，沿途有异族 |
| 雨水收集 | 雨季可补充，旱季无保障 |
| 地下水 | 少量开采，深度有限 |

**水危机**：三环外的居民每天只能获得约1-2升水（饮用+洗漱），远低于生存标准。缺水导致的疾病在三环外肆虐。

#### 6. 人力资源

**共鸣者 vs 非共鸣者**：
- 共鸣者占总人口约0.5-1%（且在下降，因为古物减少导致觉醒率降低）。以长安城80-100万人口计，共鸣者约4000-10000人，其中一阶触物者占70%以上，三阶以上不足百人
- 共鸣者是聚集地的核心战力和技术骨干
- 非共鸣者承担农业、建筑、后勤等基础工作

**人力衰减**：
- 异族消耗战持续猎杀外出的高阶共鸣者——每损失一名三阶以上共鸣者，都需要数十年才能培养出替代者
- 疾病、饥饿、异族袭击导致底层人口死亡率居高不下
- 三环外的人口虽然膨胀，但有效劳动力（健康成年男性）比例在下降

**生产型共鸣者的系统性短缺（隐藏危机）**：
- **司农堂（农业共鸣者）**：长安城粮食危机的核心原因不是没有耕地，而是没有足够的农业共鸣者——有司农堂共鸣者指导的农田产量是普通农田的3-5倍。但司农堂被武道派视为"种地的"，地位低下、资源分配最少，人数过去30年减少了60%
- **修造堂（修复共鸣者）**：城墙力场需要修造堂共鸣者持续维护。人手不足→城墙力场裂缝加速→防御衰退速度超出预判。匠造协会将最优秀的造物型共鸣者优先分配给武器炼制，进一步加剧短缺
- **医工堂（治愈共鸣者）**：三环外几乎没有医工堂覆盖，底层人的伤病只能靠草药和运气。医工堂集中内城为权贵服务——阶层冲突的又一矛盾点

### 资源冲突矩阵

| 冲突类型 | 驱动资源 | 表现形式 |
|----------|----------|----------|
| 阶层冲突 | 食物、水、盐 | 三环外暴动、三环内外对立、缺盐导致的大规模健康危机 |
| 世家之争 | 古物、共鸣者人才 | 公会内部派系斗争、世家联姻与暗杀 |
| 拾荒者冲突 | 矿物、末世前物资、废墟开路权 | 三环外拾荒队之间的地盘争夺、硝石矿脉争夺 |
| 内外冲突 | 所有资源 | 共鸣者征召制引发的不满、逃亡事件 |
| 战斗vs生产 | 共鸣者人才分配 | 武道派垄断优质人才→生产型共鸣者短缺→粮食/防御/医疗危机 |
| 代际冲突 | 觉醒机会 | 年轻人觉醒率下降引发的社会焦虑 |
| 私盐/私药冲突 | 盐、药材 | 三环外私盐贩子与公会盐务管控的博弈、底层对成品药的绝望需求 vs 公会垄断 |
| 火药管控 | 硝石/硫磺 | 匠造协会垄断火药配方和品质管控 vs 拾荒者对爆破开路的刚需、黑市火药交易 |

### 资源衰减的时间线预判

如果不打破异族封锁线，长安城的资源储备预判如下（文心派基于残存文献的粗略推算，误差极大，但趋势方向可信）：

按当前消耗速率推算，二三十年内长安城将面临全面崩溃。关键节点的预估方向如下（具体年份不可靠，但崩溃顺序大致如此）：

- **近期（当前-10年内）**：粮食缺口持续扩大，古物死寂加速，三环外暴动频发。盐和基础药材的短缺开始引发三环外的大规模健康危机——缺盐导致的碘缺乏症和肌肉痉挛在三环外蔓延，因伤口感染（缺药）死亡的拾荒者数量持续上升
- **中期（10-20年内）**：近处废墟资源基本耗尽，三环外可能出现大规模饥荒。共鸣者培养周期进一步延长，高阶共鸣者面临断代。盐源可能完全枯竭，食物保存能力崩溃——即使有粮食也无法储存，饥荒将加速
- **远期（20-30年内）**：城墙力场因古物死寂而显著减弱，聚集地防御面临崩溃风险。人类进入最终倒计时

**这就是异族"围而不攻"的恐怖之处**——它不需要进攻，只需要等。人类在有限资源中自我消耗，最终不攻自破。打破围困、获取新资源，是人类生存的唯一出路。
