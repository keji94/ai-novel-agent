## 七、社会结构

### 聚集地通用结构

```
          ┌─────────┐
          │ 核心区域  │ ← 古建筑本体所在，权贵居住区
          │ （内城）  │    最强防御，资源最丰富
          └────┬────┘
               │
       ┌───────┴───────┐
       │   二环区域     │ ← 中产/共鸣者居住区
       │   （商住区）    │    有基础防御设施
       └───────┬───────┘
               │
       ┌───────┴───────┐
       │   三环区域     │ ← 底层/无共鸣者居住区
       │   （贫民区）    │    防御薄弱，随时可能被放弃
       └───────┴───────┘
               │
        ┌──────┴──────┐
        │  三环外废墟  │ ← 拾荒者、流浪者
        │  （荒野区）   │    异族出没，朝不保夕
        └─────────────┘
```

注：三环结构是长安城的典型模式，其他聚集地的社会结构各不相同（详见《世界观设定-聚集地格局》）。

### 120年演变下的社会特征
- **阶层固化**：经过几代人的发展，共鸣者世家已经形成。高阶共鸣者的后代觉醒概率更高，"血统论"深入人心。三环内的家族已把持权力数十年，底层几乎无上升通道
- **三环鸿沟**：三环内外不只是贫富差距，而是两个完全不同的世界。三环内的人可能一辈子没见过异族长什么样；三环外的人从记事起就在为生存挣扎。两边的价值观、语言习惯甚至信仰都已有明显差异
- **知识垄断**：共鸣者公会有意无意地垄断着关于古物共鸣的系统知识。文心派的书卷共鸣者掌握了大量末世前的残存文献，但他们只传授给"有资格"的人。底层人对古物的认知还停留在"神器崇拜"的阶段
- **资源焦虑**：随着资源持续衰减，"资源焦虑"已成为聚集地社会的底色。三环内担心古物枯竭、城墙力场减弱；三环外担心粮食和水源。这种焦虑是所有社会矛盾的燃料

### 长安城核心势力

**1. 共鸣者公会**
- 聚集地的实际掌控者
- 由高阶共鸣者组成，历经数代已形成世家格局
- 负责对外征伐、内部秩序
- 等级森严，低阶服从高阶
- 内部派系林立，暗斗不断

**2. 匠造协会**
- 炼制古物衍生品的专业组织
- 地位超然，各势力都需要的合作对象
- 内部有严格的师徒传承制度
- 掌握着大部分"实用技术"的知识，是末世中技术传承的核心力量
- 负责城墙力场的维护——这是他们在长安城的立身之本

**3. 商会**
- 掌控聚集地的经济命脉
- 收购古物、出售物资
- 有自己的武装力量
- 部分商会家族的历史可追溯到复兴纪元初期
- 在资源衰减的大背景下，商会的囤积居奇行为加剧了底层的不满

**4. 拾荒者**
- 三环外生存的人群
- 靠在废墟中搜寻古物和物资为生
- 主角的开局身份
- 社会地位最低，但最了解废墟
- 形成了自己的"废墟文化"和生存法则，三环内的人根本无法理解
- 拾荒者是资源衰减最直接的受害者——废墟越捡越空，他们需要走越来越远

**5. 信仰教派**
- 崇拜古物/崇拜祖先的宗教组织
- 各聚集地都有不同教派
- 有的温和、有的极端
- 120年间教派经历了多次分裂和融合，形成了复杂的信仰图谱
- 主流教派将博物馆视为"圣地"，认为其中保存着"失落文明"的终极秘密
- 在资源焦虑的社会中，教派的影响力越来越大——人们需要精神寄托

### 觉醒管控：统治者的"三锁"体系

统治者围绕共鸣的三个觉醒条件，构建了一套精密的管控体系——后世尊称为"三锁"。

**第一锁：基因之锁——控制觉醒的"原材料"**
- 源潮基因人人皆有，统治者无法筛选谁有天赋。但他们通过控制**基因的温养环境**来间接影响觉醒潜力
- 具体手段：限制平民接触高密度古物（防止基因在无准备状态下被"激活又浪费"）、控制饮食和居住环境（以"净化体质"名义实行阶层隔离）、限制三环外居民进入内城古建筑区
- 底层逻辑：不是阻止觉醒，而是**确保觉醒发生在统治者可控的范围内**

**第二锁：压力之锁——用异族的刀筛选人命**
- 觉醒需要"生死绝境中的异族压力"——这是统治者最狠也最有效的手段
- 具体手段：
  - **荒野试炼**：定期组织"先祖试炼"，将筛选出的有潜力的年轻人投入异族出没区域，名义上是勇者之路，本质是拿人命当觉醒的催化剂
  - **防线策略性收缩**：刻意放松某些外围防线的防守，让低阶异族"渗透"进三环外居住区。死一批平民没关系——**逼出几个觉醒者就是赚的**
  - **异族活体囚禁**：部分聚集地秘密饲养低阶异族，用于对"觉醒候选人"进行人工压力刺激。这种做法在各聚集地属于禁忌，但几乎人人都知道它的存在
- 美其名曰："先祖试炼""勇者的必经之路""天选之人的磨砺"
- 底层逻辑：**统治者不希望平民觉醒太多**。理想状态是每隔一段时间"产出"几个可控的低阶觉醒者，纳入体制后感恩戴德（"是聚集地救了我"）

**第三锁：知识之锁——控制共鸣的上限**
- 对古物的理解决定共鸣深度——而知识是最容易被垄断的
- 文心派天然是统治者的盟友：
  - 古物知识不公开，平民只能"崇拜"古物而无法"理解"古物
  - 即使有人在绝境中侥幸觉醒，没有知识储备也只能停留在**一阶触物者**，翻不了天
  - 高阶共鸣者全部来自"被选中的人"——统治阶层和他们的嫡系
- 匠造协会也参与知识封锁：他们垄断了古物衍生品的炼制技术，平民即使得到古物碎片也不知道如何使用
- 底层逻辑：**控制知识就控制了共鸣的天花板**。一个一阶触物者再愤怒，也对抗不了三阶以上的体制共鸣者

**三锁的协同效应**：
```
平民的一生（在统治者眼中）：
  出生（基因锁：被安排在三环外）→ 成长（知识锁：接触不到真正的古物知识）
  → 被选中参加试炼（压力锁：扔进荒野）→ 侥幸觉醒（通常只有一阶）
  → 被征召入体制（感恩戴德）→ 成为统治者的工具
```

**三锁的裂缝**：
- 拾荒者在废墟中偶尔会接触到真正的古物和末世前文献——这是三锁覆盖不到的灰色地带
- 跨聚集地的流浪者和商人会带来知识碎片，虽然不成体系，但足以让有心人产生怀疑
- 部分信仰教派以"先祖启示"的名义保存和传播着被扭曲的古物知识——虽然是错的，但偶尔歪打正着
- **主角苏辰的突破路径**：来自末世前的完整知识储备让他直接绕过了第三锁；而前世的经历和意志让他能主动寻找异族压力（绕过第二锁）；源潮基因本身人人都有（第一锁无效）。三锁对他形同虚设——这就是金手指的世界观底层支撑

### 货币体系
- **源晶**：异族体内凝结的能量结晶，用途广泛。是最通用的硬通货
- **古物碎片**：古物碎裂后的残余，可用于低级共鸣催化。价值低于源晶但需求稳定
- **以物易物**：在三环外仍然普遍，粮食、水源、药品都可以直接交换
