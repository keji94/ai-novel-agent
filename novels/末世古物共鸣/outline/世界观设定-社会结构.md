## 七、社会结构

### 聚集地通用结构

```
          ┌─────────┐
          │ 核心区域  │ ← 古建筑本体所在，权贵居住区
          │ （内城）  │    最强防御，资源最丰富
          └────┬────┘
               │
       ┌───────┴───────┐
       │   二环区域     │ ← 中产/共鸣者居住区
       │   （商住区）    │    有基础防御设施
       └───────┬───────┘
               │
       ┌───────┴───────┐
       │   三环区域     │ ← 底层/无共鸣者居住区
       │   （贫民区）    │    防御薄弱，随时可能被放弃
       └───────┴───────┘
               │
        ┌──────┴──────┐
        │  三环外废墟  │ ← 拾荒者、流浪者
        │  （荒野区）   │    异族出没，朝不保夕
        └─────────────┘
```

注：三环结构是长安城的典型模式，其他聚集地的社会结构各不相同（详见《世界观设定-聚集地格局》）。

### 120年演变下的社会特征
- **阶层固化**：经过几代人的发展，共鸣者世家已经形成。高阶共鸣者的后代觉醒概率更高，"血统论"深入人心。三环内的家族已把持权力数十年，底层几乎无上升通道
- **三环鸿沟**：三环内外不只是贫富差距，而是两个完全不同的世界。三环内的人可能一辈子没见过异族长什么样；三环外的人从记事起就在为生存挣扎。两边的价值观、语言习惯甚至信仰都已有明显差异
- **知识垄断**：共鸣者公会有意无意地垄断着关于古物共鸣的系统知识。文心派的书卷共鸣者掌握了大量末世前的残存文献，但他们只传授给"有资格"的人。底层人对古物的认知还停留在"神器崇拜"的阶段
- **资源焦虑**：随着资源持续衰减，"资源焦虑"已成为聚集地社会的底色。三环内担心古物枯竭、城墙力场减弱；三环外担心粮食和水源。这种焦虑是所有社会矛盾的燃料

### 统治者的核心困境：觉醒者数量与权力的悖论

聚集地的统治者面临一个根本性的矛盾：**他们需要觉醒者来维持生存，又害怕觉醒者动摇统治。**

异族不会因为人类内部分裂就停止进攻。城墙力场在衰减，古物在枯竭，每一波虫潮都比上一波更难抵挡。统治者比谁都清楚——光靠世家子弟的觉醒者，根本不够填这个窟窿。他们必须从底层"收获"新的共鸣者。

但觉醒本身就是危险的。一个在生死之间觉醒的人，获得了足以对抗体制的力量，凭什么还要对把他扔进荒野等死的统治者感恩戴德？更麻烦的是，觉醒不可逆、不可预测——你不知道谁会在什么时候觉醒，觉醒到什么程度。一个不受控制的二阶共鸣者，足以成为世家权力的威胁。

120年来，统治者在刀尖上摸索出了一套策略，核心思路不是"压制觉醒"，而是"**收割觉醒**"——让觉醒发生，但确保它发生在统治者划定的框架内。

**实际策略：**

- **定向筛选**：从三环外挑选"有潜力"的年轻人（通常是体格健壮、家族无背景的孤身者），纳入"先祖试炼"体系。这些人即使觉醒，也缺乏社会关系，容易被收编
- **隔离孵化**：觉醒者一旦被发现，立刻从原有社会关系中剥离，编入共鸣者公会的体系。切断他与三环外旧友、家人的联系，用资源和地位重新塑造他的忠诚
- **阶梯式授权**：不一次给足力量。新觉醒者先在公会内以"学徒"身份服役数年，能力培养和忠诚考核并行。高阶古物的使用权限严格按等级分配，即使天赋出众，没有"履历"也接触不到真正强大的古物
- **内部分化**：公会内部刻意制造派系竞争。让新晋觉醒者依附于某个世家派系，用资源分配拉拢，用晋升通道控制。觉醒者之间彼此牵制，就不太可能联合起来挑战体制

**这套策略的漏洞：**

- 资源衰减让"阶梯式授权"越来越难维持——公会已经拿不出足够的筹码来收编新觉醒者了。一个觉醒后发现自己只能拿到破铜烂铁的年轻人，和被扔进荒野前有什么区别？
- 世家的"定向筛选"越来越依赖血统论，但共鸣的触发条件（异族压力、古物理解）与血统并无直接关联。不少世家子弟反复试炼仍无法觉醒，而三环外偶尔蹦出的觉醒者却被当作"意外"处理掉——这种双重标准正在被越来越多的人注意到
- 隔离孵化需要时间，但虫潮不等人。战事吃紧时，公会不得不加快收编节奏，放松忠诚考核。一些立场不稳的觉醒者就这样混进了体制内部

这些裂缝不会立刻撕裂社会，但它们是暗流——主角将在这片暗流中找到自己的空间。

### 长安城核心势力

**1. 共鸣者公会**
- 聚集地的实际掌控者
- 由高阶共鸣者组成，历经数代已形成世家格局
- 负责对外征伐、内部秩序
- 等级森严，低阶服从高阶
- 内部派系林立，暗斗不断

**2. 匠造协会**
- 炼制古物衍生品的专业组织
- 地位超然，各势力都需要的合作对象
- 内部有严格的师徒传承制度
- 掌握着大部分"实用技术"的知识，是末世中技术传承的核心力量
- 负责城墙力场的维护——这是他们在长安城的立身之本

**3. 商会**
- 掌控聚集地的经济命脉
- 收购古物、出售物资
- 有自己的武装力量
- 部分商会家族的历史可追溯到复兴纪元初期
- 在资源衰减的大背景下，商会的囤积居奇行为加剧了底层的不满

**4. 拾荒者**
- 三环外生存的人群
- 靠在废墟中搜寻古物和物资为生
- 主角的开局身份
- 社会地位最低，但最了解废墟
- 形成了自己的"废墟文化"和生存法则，三环内的人根本无法理解
- 拾荒者是资源衰减最直接的受害者——废墟越捡越空，他们需要走越来越远

**5. 信仰教派**
- 崇拜古物/崇拜祖先的宗教组织
- 各聚集地都有不同教派
- 有的温和、有的极端
- 120年间教派经历了多次分裂和融合，形成了复杂的信仰图谱
- 主流教派将博物馆视为"圣地"，认为其中保存着"失落文明"的终极秘密
- 在资源焦虑的社会中，教派的影响力越来越大——人们需要精神寄托

### 觉醒管控：统治者的"三锁"体系

统治者围绕共鸣的三个觉醒条件，构建了一套精密的管控体系——后世尊称为"三锁"。

**第一锁：接触之锁——掐断觉醒的触发条件**
- 共鸣的触发需要人与古物发生深度接触。源潮基因人人都有，但基因本身不构成差异——真正的变量在于**接触古物的机会和质量**
- 具体手段：
  - 内城古建筑区实行严格的出入管制，平民未经许可不得踏入核心区域。这直接切断了底层接触高密度古物的途径
  - 三环外的废墟被划分为"官方拾荒区"和"禁区"。禁区往往是古建筑密集区，名义上是因为"异族出没太危险"，实际上是因为那里有触发共鸣的潜力
  - 平民日常能接触到的古物碎片被刻意筛选过——低纯度、低共鸣潜力，不足以触发觉醒
- 底层逻辑：不是阻止觉醒，而是**确保觉醒发生在统治者可控的范围内**。你在三环外捡一辈子破铜烂铁也醒不了，但如果你被选中进入内城……那一切就不同了

**第二锁：压力之锁——用异族的刀筛选人命**
- 觉醒需要"生死绝境中的异族压力"——这是统治者最狠也最有效的手段
- 具体手段：
  - **荒野试炼**：定期组织"先祖试炼"，将筛选出的有潜力的年轻人投入异族出没区域，名义上是勇者之路，本质是拿人命当觉醒的催化剂
  - **封锁线缝隙**：外围封锁线并非铁板一块——部分偏远区段的巡逻密度长期不足，低阶异族偶尔会从缝隙中渗透进三环外居住区。这不是疏忽，而是一种冷酷的成本计算：死一批平民没关系，**逼出几个觉醒者就是赚的**
  - **虫卵压力实验**：极少数聚集地秘密进行的高风险操作——获取虫潮产下的未孵化虫卵，在受控环境中让虫卵对"觉醒候选人"施加生物压力。虫卵在孵化前处于无意识状态，不会主动攻击，但其源潮辐射对人类基因有刺激作用。这种做法在所有聚集地都是禁忌，一旦虫卵提前孵化，后果不堪设想，因此只有最激进的势力才会冒险尝试
- 美其名曰："先祖试炼""勇者的必经之路""天选之人的磨砺"
- 底层逻辑：**统治者不希望平民觉醒太多**。理想状态是每隔一段时间"产出"几个可控的低阶觉醒者，纳入体制后感恩戴德（"是聚集地救了我"）

**第三锁：知识之锁——控制共鸣的上限**
- 对古物的理解决定共鸣深度——而知识是最容易被垄断的
- 知识管控的具体手段：
  - 古物知识不公开，平民只能"崇拜"古物而无法"理解"古物。文心派掌握着末世前残存文献的系统解读权，但只传授给"被选中的人"
  - 即使有人在绝境中侥幸觉醒，没有知识储备也只能停留在**一阶触物者**，翻不了天
  - 匠造协会也参与知识封锁：他们垄断了古物衍生品的炼制技术，平民即使得到古物碎片也不知道如何使用
  - 公会内部对古物知识的传播有严格的等级门槛——二阶共鸣者才有资格接触"共鸣原理"，三阶以上才能研读"古物本源"
- 底层逻辑：**控制知识就控制了共鸣的天花板**。一个一阶触物者再愤怒，也对抗不了三阶以上的体制共鸣者

**三锁的协同效应**：
```
平民的一生（在统治者眼中）：
  出生（接触锁：被限制在三环外，接触不到真正的古物）→ 成长（知识锁：学不到共鸣知识）
  → 被选中参加试炼（压力锁：扔进荒野）→ 侥幸觉醒（通常只有一阶）
  → 被征召入体制（感恩戴德）→ 成为统治者的工具
```

**三锁的裂缝**：
- 拾荒者在废墟中偶尔会接触到真正的古物和末世前文献——这是三锁覆盖不到的灰色地带
- 跨聚集地的流浪者和商人会带来知识碎片，虽然不成体系，但足以让有心人产生怀疑
- 部分信仰教派以"先祖启示"的名义保存和传播着被扭曲的古物知识——虽然是错的，但偶尔歪打正着
- 那些在三锁夹缝中觉醒却未被体制收编的人，是最不稳定的变量。他们有力量，有怨恨，无处可去——他们是暗流中最危险的水

### 货币体系
- **源晶**：异族体内凝结的能量结晶，用途广泛。是最通用的硬通货
- **古物碎片**：古物碎裂后的残余，可用于低级共鸣催化。价值低于源晶但需求稳定
- **以物易物**：在三环外仍然普遍，粮食、水源、药品都可以直接交换
