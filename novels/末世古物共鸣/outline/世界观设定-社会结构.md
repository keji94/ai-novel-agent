## 七、社会结构

### 聚集地通用结构

```
          ┌─────────┐
          │ 核心区域  │ ← 古建筑本体所在，权贵居住区
          │ （内城）  │    最强防御，资源最丰富
          └────┬────┘
               │
       ┌───────┴───────┐
       │   二环区域     │ ← 中产/共鸣者居住区
       │   （商住区）    │    有基础防御设施
       └───────┬───────┘
               │
       ┌───────┴───────┐
       │   三环区域     │ ← 底层/无共鸣者居住区
       │   （贫民区）    │    防御薄弱，随时可能被放弃
       └───────┴───────┘
               │
        ┌──────┴──────┐
        │  三环外废墟  │ ← 拾荒者、流浪者
        │  （荒野区）   │    异族出没，朝不保夕
        └─────────────┘
```

注：三环结构是长安城的典型模式，其他聚集地的社会结构各不相同（详见第五章"聚集地孤岛格局"）。

### 120年演变下的社会特征
- **阶层固化**：经过几代人的发展，共鸣者世家已经形成。高阶共鸣者的后代觉醒概率更高，"血统论"深入人心。三环内的家族已把持权力数十年，底层几乎无上升通道
- **三环鸿沟**：三环内外不只是贫富差距，而是两个完全不同的世界。三环内的人可能一辈子没见过异族长什么样；三环外的人从记事起就在为生存挣扎。两边的价值观、语言习惯甚至信仰都已有明显差异
- **知识垄断**：共鸣者公会有意无意地垄断着关于古物共鸣的系统知识。文心派的书卷共鸣者掌握了大量末世前的残存文献，但他们只传授给"有资格"的人。底层人对古物的认知还停留在"神器崇拜"的阶段
- **资源焦虑**：随着资源持续衰减，"资源焦虑"已成为聚集地社会的底色。三环内担心古物枯竭、城墙力场减弱；三环外担心粮食和水源。这种焦虑是所有社会矛盾的燃料

### 长安城核心势力

**1. 共鸣者公会**
- 聚集地的实际掌控者
- 由高阶共鸣者组成，历经数代已形成世家格局
- 负责对外征伐、内部秩序
- 等级森严，低阶服从高阶
- 内部派系林立，暗斗不断

**2. 匠造协会**
- 炼制古物衍生品的专业组织
- 地位超然，各势力都需要的合作对象
- 内部有严格的师徒传承制度
- 掌握着大部分"实用技术"的知识，是末世中技术传承的核心力量
- 负责城墙力场的维护——这是他们在长安城的立身之本

**3. 商会**
- 掌控聚集地的经济命脉
- 收购古物、出售物资
- 有自己的武装力量
- 部分商会家族的历史可追溯到复兴纪元初期
- 在资源衰减的大背景下，商会的囤积居奇行为加剧了底层的不满

**4. 拾荒者**
- 三环外生存的人群
- 靠在废墟中搜寻古物和物资为生
- 主角的开局身份
- 社会地位最低，但最了解废墟
- 形成了自己的"废墟文化"和生存法则，三环内的人根本无法理解
- 拾荒者是资源衰减最直接的受害者——废墟越捡越空，他们需要走越来越远

**5. 信仰教派**
- 崇拜古物/崇拜祖先的宗教组织
- 各聚集地都有不同教派
- 有的温和、有的极端
- 120年间教派经历了多次分裂和融合，形成了复杂的信仰图谱
- 主流教派将博物馆视为"圣地"，认为其中保存着"失落文明"的终极秘密
- 在资源焦虑的社会中，教派的影响力越来越大——人们需要精神寄托

### 货币体系
- **源晶**：异族体内凝结的能量结晶，用途广泛。是最通用的硬通货
- **古物碎片**：古物碎裂后的残余，可用于低级共鸣催化。价值低于源晶但需求稳定
- **以物易物**：在三环外仍然普遍，粮食、水源、药品都可以直接交换
