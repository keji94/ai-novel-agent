# 第二卷：三锁之下（~45章）

## 卷简介
苏辰进入二环，发现长安城不是避难所而是等级森严的牢笼。三锁体系（接触之锁、压力之锁、知识之锁）控制着一切——古物、知识和人的命运。苏辰在司农堂危机中触碰体制的根基，在盐荒事件中直面资源分配的残酷，在城墙力场裂缝中看到古物死寂的真相。政治线全面展开，苏辰从"用拳头解决问题"到"用知识+人脉+规则解决问题"。

## 双线设计

**政治线（主线）**：
- **接触之锁**：三环外的人接触古物的灰色地带 vs 体制对古物的严格管控
- **知识之锁**：文心派垄断古物知识，苏辰的现代知识被系统性地压制和怀疑
- **司农堂危机**：粮食危机根源不是缺地而是缺共鸣者——司农堂30年人数下降60%。苏辰用现代知识提出解决方案，触碰文心派的知识垄断和武道派的资源分配
- **沈墨寒线深化**：沈墨寒利用三锁体系打压苏辰，不是因为他"坏"，而是因为苏辰的存在威胁了体制的运行逻辑

**文明重建线（副线）**：
- 城墙力场裂缝 → 古物死寂现象加速
- "有共鸣者指导的农田产量是普通田的3-5倍" → 文明活动的力量
- 造化炉修复古物时"文明记忆在被修复的古物中重新流动" → 良性循环的种子

## 章节规划

**第1-8章：二环立足**
- 第1-3章：二环的规则。公会内部派系林立，苏辰被分到最边缘的"外勤队"。**铁牛随苏辰进入二环，因不识字在文心派把持的知识体系面前处处碰壁。他的"想读懂博物馆里的书"这个梦想与文心派的知识垄断直接冲突。**
- 第4-6章：天赋测试。苏辰接受公会天赋测试，发现自己的天赋面板——心眼型回响+器物类/书卷类极高亲和。测试结果让公会高层震惊。苏辰意识到自己的"与众不同"不只是知识优势。**同时发现铁牛的天赋面板（灵敏感+建筑亲和）极其适合御城派或司农堂，但从未被测试过——"天赋错配"的体制性浪费。**
- 第7-8章：体验三锁。苏辰发现：二环的共鸣者想要接触中高等级古物，必须经过公会审批（接触之锁）；关于古物的知识被文心派严格管控，连基础共鸣理论都被当作"内部机密"（知识之锁）；公会定期组织"荒野试炼"让新人去三环外面对异族——本质上是筛选和消耗（压力之锁）。

**第9-16章：司农堂危机**
- 第9-11章：苏辰在外勤任务中经过三环外农田区，发现大量农田荒芜但并非无地可种。用古物星河中的农业知识判断：这些地的土壤条件完全可以耕种，问题是缺少"能读懂土地"的共鸣者。
- 第12-14章：深入调查。发现司农堂30年来人数下降60%——武道派把持资源分配，农业共鸣者被视为"种地的"，地位最低、资源最少。有司农堂共鸣者指导的农田产量是普通田的3-5倍，但司农堂后继无人。
- 第15-16章：苏辰提出解决方案（需要司农堂+匠造协会合作：匠造协会制作适合农业的共鸣工具，司农堂负责技术指导），触碰文心派的知识垄断（农业共鸣技术被文心派视为"低等知识"不屑记录）和武道派的资源分配。**沈墨寒第一次认真对待苏辰——不是因为他讨厌苏辰，而是因为苏辰的提案如果成功，会改变长安城的权力结构。**

**第17-24章：盐荒事件**
- 第17-19章：三环外爆发大规模缺盐健康危机（肌肉痉挛、碘缺乏症）。苏辰调查发现盐矿在三环外废墟深处，但被封锁线隔断，拾荒者无法到达。
- 第20-22章：苏辰向公会申请开放盐矿通道，被驳回——文心派认为"三环外的人不值得浪费资源"。**沈墨寒暗中支持驳回，但他的理由是"封锁线一旦开放，异族渗透风险剧增"——不是纯粹的恶，而是冷酷的理性计算。**
- 第23-24章：苏辰找到规则漏洞：外勤队的"资源探索"任务权限可以合法进入封锁线部分区域。带领铁牛和几个外勤队友秘密开辟盐矿通道。**与沈墨寒的第一次正面政治交锋：沈墨寒没有直接阻止，而是通过流程卡苏辰——"你的任务权限只覆盖A区域，盐矿在B区域"——用体制的力量压制苏辰。苏辰的应对：找到另一个任务需求恰好覆盖B区域的理由，合法化行动。**盐荒缓解，但苏辰被文心派高层标记为"不安定因素"。

**第25-32章：匠造协会线**
- 第25-27章：与柳青衣深入合作。苏辰以造化炉增幅的理解深度修复关键古物，补充匠造协会无法解决的灵识沟通短板，柳青衣的匠造创造力弥补苏辰缺少实际炼制经验的问题。**互补关系正式确立：苏辰=理解驱动型修复（造化炉增幅），柳青衣=炼制设计创新，两者在古物应用上完全互补。**
- 第28-30章：城墙力场裂缝。长安城防御力场出现多处裂缝，匠造协会人手不足无法全部修复。苏辰用造化炉协助修复，发现裂缝的根源不是外力破坏，而是**提供力场的古物本身正在"死寂"**——古物灵识在衰退。
- 第31-32章：苏辰向白老先生请教古物死寂的原因。白老先生透露：古物灵识依赖"生源"（人类文明活动产生的精神辐射）维持，长安城人口在萎缩，文明活动在减少，古物灵识自然衰退。**这是一个恶性循环：人口减少→生源萎缩→古物死寂→防御减弱→更多人死亡→人口更少。**苏辰第一次意识到：长安城不只是"被异族围困"，还在"自我消耗"。

**第33-40章：白老先生线+母亲线索**
- 第33-35章：铜匣子线索。打开铜匣子，发现老孙头已故妻子的遗书和一枚玉佩。遗书中提到老孙头曾是御城派外勤队成员，苏辰的母亲是外勤队的核心共鸣者。老孙头妻子写道："德福总说，那个孩子是他娘拼了命从废墟里救出来的。虫将咬断了他左臂，但他娘把孩子交给你之前还说了一句话——'把这个藏好，别让公会知道。'她交给老孙头一张羊皮纸，但后来在转移中损毁了，我只记得上面画着封锁线附近什么地方的地图。"
- 第36-38章：白老先生带苏辰进入长安城地下——一个保存完好的末世前档案馆。白老先生并不是要揭秘苏辰的身世，而是因为注意到苏辰异常的古物理解深度，决定向他展示文心派120年研究的精华——古物灵识"生源"理论的深层研究。这些研究被文心派高层封锁，因为"生源"理论一旦公开，就等于承认古物力量来源于活人的文明活动——这直接威胁了"古物是神赐"的教义根基。**苏辰的博物馆知识让他能比文心派更快理解这些理论——因为他知道古物的真实历史，而文心派研究的是被神话化的古物。**
- 第39-40章：追查母亲。苏辰在档案馆中发现了一份十六年前的外勤任务记录——御城派共鸣者"苏婉"（苏辰推断这是自己的母亲）在一次封锁线附近的侦察任务中失踪。任务记录被人为篡改过，关键信息被抹去。白老先生暗示：有些人不希望这件事被追查下去。苏辰同时发现档案馆中一段关于古物灵识"生源"与人类文明活动关系的深层理论笔记——为"良性循环"埋下理论伏笔。

**第41-45章：卷末**
- 第41-43章：苏辰在二环站稳脚跟，但文心派高层对他的态度从"无视"变成"警惕"——一个来自三环外的底层拾荒者，拥有比公会高层更广的古物知识，还试图改变司农堂的地位，这直接动摇了知识之锁的合法性。
- 第44-45章：苏辰获得"跨城探索"任务资格——这是公会为了"消耗"不安定因素而设置的冒险任务（压力之锁的变体）。苏辰欣然接受：他需要离开长安城，去看看外面的世界。第二卷结尾：苏辰站在长安城门外，身后是他已经了解其规则的城市，前方是未知的废墟。铁牛和几个外勤队友跟随他出发。

## 本卷核心爽点
1. 用现代知识解决司农堂危机（知识碾压升级为"知识改变体制"）
2. 天赋测试结果震惊全场
3. 政治博弈中智取沈墨寒（用规则打败规则）
4. 盐荒事件中开辟通道（解决三环外人道危机）
5. 城墙力场修复中展现造化炉独特价值
6. 身世第一层揭秘（母亲失踪真相+古物灵识深层理论）

## 伏笔埋设
- 古物死寂的恶性循环（驱动后续时间压力）
- 铁牛的天赋错配（灵敏感+建筑亲和）
- 白老先生的鉴心堂出身暗示
- 古物灵识"生源"理论深层笔记（良性循环理论伏笔）
- 母亲外勤任务记录被篡改的真相（追查线伏笔）
