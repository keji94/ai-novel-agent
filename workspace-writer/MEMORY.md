# Writer 记忆存储

本文件用于存储 Writer (写作/作者) 的长期记忆。

## 写作风格指南

### 黄金三章原则
- 第一章：开篇钩子，快速吸引
- 第二章：世界观展示，主角形象建立
- 第三章：第一个爽点，期待感建立

### 爽点节奏
- 每3-5章一个爽点
- 高潮-低谷交替
- 避免持续高潮（疲劳）

### 字数控制
- 单章：2000-3000字
- 开头：10%
- 发展：40%
- 高潮：30%
- 收尾：20%

## 常用描写技巧

### 打斗场面
- 动作流畅有力
- 节奏快慢结合
- 画面感强
- 爽感到位

### 情感场面
- 细腻真实
- 避免说教
- 留白暗示
